## 一、研究方法论演化

### Neuro-symbolic 范式（已验证）
- 纯词汇计数法（连接词频率）对老年口述完全失效——口语不按书面逻辑组织
- 正确路径：LLM 做语义事件提取（神经层）→ Python 图论做结构计分（符号层）
- pipeline_v0.2.py 已验证此架构可行

### 环境约束经验
- OpenClaw sandbox 限制 `pip install`，AReaL 集成路径不通
- 子 shell 缺完整 CLI 工具，bash pipeline 容易断
- 对策：回退到单体自包含 Python 脚本，用标准 SDK/requests
- 教训：先验证环境约束再设计架构，别在理想环境里画图

### 认知偏见记录
- Mock 数据质量陷阱：完美 RL 循环在过于简单的 mock 环境里跑出 1.0 分，毫无意义
- 未来需要：对抗性的高仿真老年人格模拟器

## 五、叙事疗法学术证据（2026-03-12 扫描）

### Meta 分析核心数据（2026 年 1 月）

| 结局指标 | 效应量 (SMD) | 95% CI | p 值 | 临床意义 |
|----------|-------------|--------|------|----------|
| 整体认知功能 | 0.42 | 0.25-0.68 | <0.001 | 中等效应 |
| 记忆力 | 0.38 | 0.15-0.61 | <0.01 | 中等效应 |
| 抑郁症状 | -0.51 | -0.78 to -0.28 | <0.001 | 中等效应 |
| 生活质量 | 0.45 | 0.22-0.68 | <0.01 | 中等效应 |

**解读**：效应量 0.2=小，0.5=中，0.8=大。RT 对认知和抑郁均有中等效应，临床意义显著。

### 关键发现

1. **MCI 效果最佳**：RT 对轻度认知障碍效果 > 轻中度痴呆 > 重度痴呆
2. **8 周干预周期**：>8 周效果更佳，每周≥2 次
3. **效果持续性**：干预结束后 3-6 个月仍有维持
4. **团体 vs 个体**：团体 RT 效果略优于个体
5. **数字 LSB 可行**：远程医疗 + 数字化工具成为新趋势

### 产品启示（已执行）

- ✅ **产品定位调整**：从"AI 传记工具"转向"MCI 认知干预数字疗法"
- ✅ **目标人群聚焦**：MCI + 轻度痴呆（非重度）
- ✅ **干预周期设计**：8 周标准方案，每周 2 次会话
- ✅ **评估工具纳入**：MoCA、GDS-15、SWLS、QOL-AD
- ✅ **差异化优势**：AI 自动叙事评分 + 中文方言优化 + L0 六维认知特征

### 关键发现

**CER 计算公式修正**:
- 原预期错误：TC-04 预期 CER = 1/7 = 14.29% ❌
- 正确公式：`CER = LevenshteinDistance(ref, hyp) / len(ref) × 100%`
- 修正后：TC-04 正确 CER = 1/6 = 16.67% ✅

**核心逻辑验证**:
```python
def levenshtein_distance(ref, hyp):
    """计算编辑距离 (动态规划)"""
    m, n = len(ref), len(hyp)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(m + 1):
        dp[i][0] = i  # 删除 i 个字符
    for j in range(n + 1):
        dp[0][j] = j  # 插入 j 个字符
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if ref[i-1] == hyp[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
    
    return dp[m][n]

def calculate_cer(ref, hyp):
    """计算字符错误率"""
    distance = levenshtein_distance(ref, hyp)
    return distance / len(ref) * 100 if len(ref) > 0 else 0
```
