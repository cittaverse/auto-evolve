## 表单逻辑设置

### 筛选逻辑

| 条件 | 结果 |
|------|------|
| 年龄 < 60 岁 | 提示"抱歉，本研究仅招募 60 岁以上长辈"，提交后标记为"不符合" |
| 智能手机使用 = "不使用" | 提示"本研究需要使用智能手机，抱歉"，标记为"不符合" |
| 微信使用 = "不使用" | 提示"本研究需要使用微信，抱歉"，标记为"不符合" |
| 愿意分享故事 = "不太愿意"或"完全不愿意" | 标记为"低意愿"，需电话确认 |
| 知情同意未全部勾选 | 无法提交 |

### 优先级标记

| 条件 | 优先级 |
|------|--------|
| 年龄 60-75 岁 + 熟练使用智能手机 + 非常愿意分享 | P0（优先） |
| 年龄 75+ 岁 + 基本会用智能手机 + 比较愿意分享 | P1（正常） |
| 需要家属协助 + 愿意分享 | P1（正常） |
| 年龄<60 岁 或 不使用智能手机 | 不符合 |

## 表单平台选择

### 腾讯问卷（推荐）
- **优点**：免费、易用、支持逻辑跳转、可生成二维码
- **缺点**：高级功能需付费
- **链接**：https://wj.qq.com/

### 金数据
- **优点**：功能强大、支持 API、数据导出方便
- **缺点**：免费版有限制
- **链接**：https://jinshuju.net/

### 问卷星
- **优点**：知名度高、功能全面
- **缺点**：免费版有广告
- **链接**：https://www.wjx.cn/
