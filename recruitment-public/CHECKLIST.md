## 阶段 6：报名处理（⏳ 待执行）

### 每日流程

- [ ] **09:00** 检查 overnight 报名
- [ ] **10:00** 导出报名数据
- [ ] **11:00** 电话确认（新增报名）
- [ ] **14:00** 发送知情同意书（已确认）
- [ ] **16:00** 跟踪同意书签署
- [ ] **19:00** 更新报名状态
- [ ] **20:00** 更新 `tracking-sheet.md`

### 状态管理

| 状态 | 动作 |
|------|------|
| 待确认 | 24 小时内电话确认 |
| 已确认 | 发送知情同意书 |
| 已签署 | 邀请入组，发送研究说明 |
| 已入组 | 加入研究群，准备干预 |
| 不符合 | 礼貌告知，感谢关注 |
| 已退出 | 记录原因，感谢参与 |
