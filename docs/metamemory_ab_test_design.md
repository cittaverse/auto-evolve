## 一、测试目标

验证元记忆引导策略对老年人叙事质量的影响：

| 假设 | 预期效果 | 验证指标 |
|------|----------|----------|
| H1: 元记忆提示提升细节丰富度 | 实验组叙事细节数量 +30% | 平均细节数/条 |
| H2: 元记忆提示提升信心水平 | 实验组信心评分 +1.5 分 | 平均信心分 (1-10) |
| H3: 元记忆提示改善叙事连贯性 | 实验组连贯性评分 +20% | L0 连贯性维度分数 |
| H4: 来源监控减少记忆混淆 | 实验组来源模糊率 -40% | 来源模糊占比 |

## 二、实验设计

### 2.1 分组设计

| 组别 | 样本量 | 干预措施 | 预期招募 |
|------|--------|----------|----------|
| **对照组 (A)** | n=25 | 标准回忆提示，无元记忆 | 社区老年活动中心 |
| **实验组 (B)** | n=25 | 标准提示 + 元记忆引导 | 社区老年活动中心 |

**总样本量**: N=50  
**统计功效**: 0.80 (α=0.05, 中等效应量 d=0.5)

### 2.2 纳入/排除标准

**纳入标准**:
- 年龄 ≥60 岁
- 母语为中文
- MMSE 评分 ≥24 分 (认知功能正常或轻度受损)
- 能够独立完成 30 分钟访谈

**排除标准**:
- 中重度痴呆 (MMSE<24)
- 严重听力/视力障碍
- 近期重大生活事件 (<3 个月)

### 2.3 实验流程

```
┌─────────────────────────────────────────────────────────┐
│                    招募与筛选 (D1-D3)                    │
│  - 社区宣传 → 报名 → MMSE 筛查 → 随机分组                │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    基线评估 (D4)                         │
│  - 人口学信息收集                                        │
│  - 基线认知评估 (MMSE, MoCA)                             │
│  - 基线叙事任务 (无干预，5 分钟自由回忆)                  │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    干预阶段 (D5-D11)                     │
│  对照组：标准提示 ("请回忆...")                          │
│  实验组：标准提示 + 元记忆引导                           │
│  - 每日 1 次，每次 15-20 分钟                              │
│  - 共 7 天，7 个主题 (童年/求学/工作/家庭等)              │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    后测评估 (D12)                        │
│  - 叙事任务 (与基线同主题，对比质量变化)                  │
│  - 主观体验问卷 (信心、满意度、难度)                      │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    数据分析 (D13-D15)                    │
│  - 叙事质量评分 (L0 系统)                                 │
│  - 组间比较 (t 检验/Mann-Whitney U)                       │
│  - 效应量计算 (Cohen's d)                                │
└─────────────────────────────────────────────────────────┘
```

## 三、干预方案

### 3.1 对照组 (Control)

**标准提示语示例**:
```
"请您回忆一下 [主题] 的经历。"
"那时候发生了什么？"
"能再多说一些吗？"
```

### 3.2 实验组 (Treatment)

**标准提示 + 元记忆引导**:

| 用户状态 | 触发条件 | 元记忆策略 | 引导语示例 |
|----------|----------|------------|------------|
| 来源模糊 | 用户说"好像是..." | 来源监控 | "这个记忆是来自您自己的经历，还是听别人说的？" |
| 信心低 | 信心评分<4/10 | 信心评估 | "您有多确定这件事是这样发生的？1-10 分打分" |
| 细节少 | 细节数<3 个 | 策略提示 | "试试按时间顺序回忆：那天早上发生了什么？然后呢？" |
| 情感丰富 | 情感词>5 个 | 意义反思 | "这件事对您来说意味着什么？" |

**元记忆 Prompt 库** (来自 `config/metamemory_prompts.yaml`):

```yaml
source_monitoring:
  - id: source_1
    question: "这个记忆是来自您自己的亲身经历，还是听别人说的？"
    rationale: "帮助区分第一手经验和二手信息"
    expected_outcome: "明确记忆来源，减少混淆"
    
  - id: source_2

  - id: confidence_2

    rationale: "促进反思"
    expected_outcome: "整合过去与现在"
```

## 四、测量指标

### 4.1 主要结局指标

| 指标 | 测量方法 | 评估时点 |
|------|----------|----------|
| **叙事细节丰富度** | 人工计数细节数量 (时间/地点/人物/事件/感官) | 基线 vs 后测 |
| **叙事质量评分** | L0 质检系统 (4 维度：Sensory/Context/Emotion/Coherence) | 基线 vs 后测 |
| **记忆信心水平** | 用户自评 (1-10 分) | 每次叙事后 |

### 4.2 次要结局指标

| 指标 | 测量方法 | 评估时点 |
|------|----------|----------|
| **来源模糊率** | 来源不明确叙事占比 | 基线 vs 后测 |
| **叙事连贯性** | L0 Coherence 维度分数 | 基线 vs 后测 |
| **用户满意度** | 5 点 Likert 量表 | 后测 |
| **任务难度感知** | 5 点 Likert 量表 | 后测 |
| **脱落率** | 未完成 7 天干预占比 | 全程追踪 |

### 4.3 探索性指标

- 不同元记忆策略的效果差异 (4 类策略对比)
- 基线认知水平对干预效果的调节作用
- 叙事主题的调节效应 (情感性 vs 中性主题)

## 五、数据分析计划

### 5.1 样本量计算

```
效应量 d = 0.5 (中等)
α = 0.05 (双尾)
Power = 0.80
→ 每组 n = 25 (G*Power 3.1 计算)
考虑 20% 脱落率 → 每组招募 31 人
```

### 5.2 统计方法

| 假设 | 统计方法 | 软件 |
|------|----------|------|
| H1: 细节丰富度 | 独立样本 t 检验 / Mann-Whitney U | Python (scipy) |
| H2: 信心水平 | 重复测量 ANOVA | Python (statsmodels) |
| H3: 连贯性改善 | 协方差分析 (ANCOVA, 控制基线) | Python (statsmodels) |
| H4: 来源模糊率 | 卡方检验 | Python (scipy) |

### 5.3 效应量计算

```python
# Cohen's d for t-test
d = (mean_treatment - mean_control) / pooled_std

# 解释标准:
# d = 0.2 → 小效应
# d = 0.5 → 中效应
# d = 0.8 → 大效应
```

### 5.4 缺失数据处理

- 脱落率 >20% → 进行敏感性分析
- 缺失值 <5% → 完整案例分析
- 缺失值 5-20% → 多重插补

## 七、伦理考量

### 7.1 知情同意

- 书面知情同意书 (包含研究目的、流程、风险、收益)
- 明确告知随时退出的权利
- 数据匿名化处理

### 7.2 风险控制

- MMSE 筛查排除中重度认知障碍
- 单次访谈不超过 30 分钟，避免疲劳
- 如出现情绪困扰，立即停止并提供支持

### 7.3 数据隐私

- 所有数据去标识化存储
- 仅研究团队访问原始数据
- 发表时仅报告汇总统计
