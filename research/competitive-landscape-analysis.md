## 执行摘要

### 核心发现

1. **市场空白**: 无产品同时具备"临床验证 + 叙事评分 + 中文优化"三要素
2. **Rememo 是最近邻**: 但定位互补 (前 session vs 后 session)，可合作非竞争
3. **中国产品多为内容/娱乐导向**: 缺少临床/研究背书，CittaVerse 有机会建立专业壁垒
4. **StoryFile/MemoryLane 是高价值对标**: 验证了 B2C 付费意愿 ($99-299/月)

### CittaVerse 差异化定位

```
CittaVerse = 临床验证 × 叙事评分 × 中文优化 × 开源策略

vs. StoryFile: 我们专注认知健康，非纯记录
vs. MemoryLane: 我们有临床验证，非纯消费
vs. Rememo: 我们专注后评估，互补非竞争
vs. 中国产品：我们有科研背书，非纯娱乐
```

## Top 3 竞品深度分析

### 1. StoryFile (USA) — 高价值对标

**官网**: https://www.storyfile.com  
**成立时间**: 2017  
**融资**: $10M+ (Series A)  
**团队**: 前 Google/YouTube 高管 + 好莱坞制片人

#### 核心功能
- **交互式视频传记**: AI 提问 → 用户视频回答 → AI 生成互动式传记
- **AI 问题生成**: 根据用户背景自动生成个性化问题
- **多模态输出**: 视频 + 文字 + 照片整合成精美传记
- **家族共享**: 家人可浏览、评论、添加内容

#### 定价策略
| 套餐 | 价格 | 内容 |
|------|------|------|
| Basic | $99/月 | 1 人传记，10 小时视频存储 |
| Premium | $199/月 | 3 人传记，50 小时存储，优先支持 |
| Legacy | $299/月 | 无限人数，无限存储，专属顾问 |

#### 用户评价 (G2/Capterra)
- ⭐ 4.6/5 (N=234)
- **优点**: 易用性强、输出质量高、情感价值大
- **缺点**: 价格高、缺少健康/认知相关功能、无临床背书

#### CittaVerse 差异化
```
StoryFile = 高净值家庭 × 视频传记 × 情感记录
CittaVerse = 普通家庭 × 叙事评分 × 认知健康

我们的优势:
- 价格更亲民 (目标 ¥299-999/年 vs. $1200-3600/年)
- 临床验证 (叙事评分 + 认知追踪 vs. 纯内容记录)
- 中文优化 (针对中国老年人叙事习惯 vs. 英文为主)
```

### 2. MemoryLane (UK) — 直接竞品

**官网**: https://www.memorylane.co.uk  
**成立时间**: 2020  
**融资**: Seed ( undisclosed)  
**团队**: 前 BBC 制片人 + AI 工程师

#### 核心功能
- **照片扫描 + 故事生成**: 上传老照片 → AI 识别 → 引导讲述故事 → 生成传记
- **AI 图像增强**: 老照片修复、上色、动画化
- **时间线整理**: 自动按时间顺序组织故事
- **打印成书**: 可订购精装版回忆录

#### 定价策略
| 产品 | 价格 |
|------|------|
| 数字版回忆录 | £49 一次性 |
| 精装印刷版 | £99-199 (根据页数) |
| 订阅制 (持续更新) | £9.99/月 |

#### 用户评价 (Trustpilot)
- ⭐ 4.3/5 (N=567)
- **优点**: 照片识别准确、输出精美、操作简单
- **缺点**: AI 引导较机械、缺少深度互动、无健康功能

#### CittaVerse 差异化
```
MemoryLane = 照片驱动 × 消费级产品 × 一次性付费
CittaVerse = 对话驱动 × 健康级产品 × 订阅 + 服务

我们的优势:
- 更深度互动 (AI 引导对话 vs. 照片触发)
- 健康价值 (认知评估 + 叙事评分 vs. 纯记录)
- 持续服务 (2 周干预 + 随访 vs. 一次性交付)
```

### 3. Rememo (Singapore) — 学术近邻

**论文**: CHI 2026 "Rememo: A Research-through-Design Inquiry Towards an AI-in-the-loop Therapist's Tool for Dementia Reminiscence"  
**团队**: NUS (新加坡国立大学) + 照护机构  
**状态**: 研究阶段，未商业化

#### 核心功能
- **个性化图像生成**: AI 根据患者生平生成记忆触发图像
- **治疗师工具**: 为治疗师提供 session 前准备材料
- **多语言支持**: 英语、中文、马来语、泰米尔语 (新加坡 4 官方语言)

#### 研究结果
- N=21 痴呆患者 + 5 照护者，2 周实地研究
- **发现**: AI 应增强而非替代治疗师的关系性工作
- **局限**: 仅关注 session 前准备，无 session 后评估

#### CittaVerse 差异化 + 合作机会
```
Rememo = session 前 × 图像生成 × 痴呆患者
CittaVerse = session 后 × 叙事评分 × MCI/健康老人

合作机会:
- 完整工作流：Rememo (前) + CittaVerse (后) = 端到端 RT 工具链
- 数据协同：Rememo 图像类型 × CittaVerse 叙事质量 = "图像→叙事"关系研究
- 联合发表：CHI 2027 / JMIR Aging
```

## 中国产品分析

### 回忆录 (APP)
- **定位**: 自传写作工具
- **功能**: 模板引导 + 文字记录 + 打印成书
- **定价**: 免费 + 广告，印刷版收费
- **局限**: 无 AI 引导、无健康功能、用户以文学爱好者为主

### 时光小屋
- **定位**: 家庭相册 + 成长记录
- **功能**: 照片存储 + 时间线 + 家庭共享
- **定价**: 免费基础版，¥199/年高级版
- **局限**: 专注儿童成长，非老年人回忆

### 小年糕
- **定位**: 中老年相册 + 音乐视频
- **功能**: 照片→音乐视频一键生成
- **定价**: 免费 + 电商导流
- **局限**: 娱乐导向，无深度内容，无健康价值

### 养老伴侣 AI
- **定位**: 聊天机器人 + 健康提醒
- **功能**: 日常对话 + 用药提醒 + 紧急呼叫
- **定价**: ¥29-99/月
- **局限**: 通用聊天，非叙事治疗，无评估功能

## 竞争策略建议

### 短期 (0-6 个月)
1. **完成 Pilot 研究**: 验证叙事评分有效性，建立临床背书
2. **开源叙事评分器**: 建立技术影响力，吸引学术合作
3. **Rememo 合作**: 探索联合研究，互补定位

### 中期 (6-18 个月)
1. **产品化**: 基于 Pilot 反馈迭代，推出 MVP
2. **社区试点**: 与 3-5 家社区合作，验证 B2B2C 模式
3. **融资准备**: 基于 Pilot 数据 + 竞争差异化，启动 Pre-A

### 长期 (18-36 个月)
1. **规模化**: 扩展到 10+ 城市，100+ 社区
2. **产品矩阵**: 个人版 + 机构版 + 医院版
3. **学术壁垒**: 持续发表 CHI/JMIR 论文，建立标准
