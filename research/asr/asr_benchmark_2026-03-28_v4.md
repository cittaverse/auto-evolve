## 2. 评估指标定义

### 2.1 WER (Word Error Rate)
```
WER = (S + D + I) / N
```
- S = Substitution (替换错误数)
- D = Deletion (删除错误数)
- I = Insertion (插入错误数)
- N = 参考文本总词数

### 2.2 CER (Character Error Rate)
```
CER = (S + D + I) / N
```
- 以汉字/字符为单位计算
- 对中文 ASR 更敏感
