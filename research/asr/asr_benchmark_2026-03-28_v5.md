## 2. 评估指标定义

### 2.1 WER (Word Error Rate)
```
WER = (S + D + I) / N
```
- S = Substitution (替换错误数)
- D = Deletion (删除错误数)
- I = Insertion (插入错误数)
- N = 参考文本总词数

### 2.2 CER (Character Error Rate)
```
CER = (S + D + I) / N
```
- 以汉字/字符为单位计算
- 对中文 ASR 更敏感

### 5.2 Mock vs 真实场景差距分析

| 测试类型 | WER | 说明 |
|----------|-----|------|
| Mock 测试 (v5) | 1.48% | 仅模拟同音/近音字错误 |
| 预期真实 ASR (标准普通话) | 5-10% | 包含噪音、口音、语速等因素 |
| 预期真实 ASR (老年语音) | 12-22% | 额外包含齿音、颤抖、停顿等 |

**关键洞察**: Mock 测试仅评估"理想条件下的文本匹配能力"，真实场景需要额外考虑:
- 环境噪音
- 录音设备质量
- 老年人生理特征 (语速慢、齿音重、颤抖)
- 方言口音
- 年代词/专有名词
