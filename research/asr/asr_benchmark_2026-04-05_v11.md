## 2. 评估指标定义

### 2.1 WER (Word Error Rate)
```
WER = (S + D + I) / N
```
- S = Substitution (替换错误数)
- D = Deletion (删除错误数)
- I = Insertion (插入错误数)
- N = 参考文本总词数

### 2.2 CER (Character Error Rate)
```
CER = (S + D + I) / N
```
- 以汉字/字符为单位计算
- 对中文 ASR 更敏感

## 5. 产品意义

### 5.1 对"一念万相"老年回忆助手的启示

1. **同音字错误是主要挑战**
   - 71.4% 的错误来自同音字替换
   - 老年人口音/语速变化可能加剧此类错误
   - 需要在后处理中加入上下文校正

2. **数字转录需要特别关注**
   - "十"与"零"的混淆可能影响关键信息 (年龄、日期、数量)
   - 建议在回忆场景中启用数字专项校验

3. **82.5% 零错误率是良好基线**
   - 在清晰发音、标准普通话场景下 ASR 表现可靠
   - 真实老年用户测试需考虑口音、语速、停顿等因素
