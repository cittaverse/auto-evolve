## 2. 评估指标定义

### 2.1 WER (Word Error Rate)
```
WER = (S + D + I) / N
```
- S = Substitution (替换错误数)
- D = Deletion (删除错误数)
- I = Insertion (插入错误数)
- N = 参考文本总词数

### 2.2 CER (Character Error Rate)
```
CER = (S + D + I) / N
```
- 以汉字/字符为单位计算
- 对中文 ASR 更敏感

## 6. 附录: 评估代码框架

```python
# WER/CER 计算工具 (待实现)
# 依赖: pip install jiwer

from jiwer import wer, cer

def calculate_metrics(reference, hypothesis):
    """
    计算 WER 和 CER
    
    Args:
        reference: 真实转录文本
        hypothesis: ASR 输出文本
    
    Returns:
        dict: {'wer': float, 'cer': float}
    """
    return {
        'wer': wer(reference, hypothesis),
```
