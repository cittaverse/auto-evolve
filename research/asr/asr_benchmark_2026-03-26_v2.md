### 1.3 测试方法
由于当前环境未配置商业 ASR API，本次测试采用 **Mock 方法** 模拟 ASR 输出，并建立完整的评估框架。

**Mock 策略**:
- 基于 AISHELL 典型文本模式生成"真实转录"(ground truth)
- 模拟常见 ASR 错误类型：
  - **同音字错误**: 玩→完、年→零、君→军、级→纪
  - **近音字错误**: 艰→坚、年→粘
- 使用标准 Levenshtein 距离算法计算 WER/CER

## 2. 评估指标定义

### 2.1 WER (Word Error Rate)
```
WER = (S + D + I) / N
```
- S = Substitution (替换错误数)
- D = Deletion (删除错误数)
- I = Insertion (插入错误数)
- N = 参考文本总词数

### 2.2 CER (Character Error Rate)
```
CER = (S + D + I) / N
```
- 以汉字/字符为单位计算
- 对中文 ASR 更敏感
