| 字段 | 值 |
|------|-----|
| **标题** | Managing emotional distress in older adults with mental illness: a randomized controlled trial evaluating virtual reality relaxation |
| **作者** | Seethaler M, Güntner L, Lütt A, Just SA |
| **DOI** | 10.1038/s41398-026-03955-1 |
| **日期** | 2026-03-20 |
| **期刊** | Translational Psychiatry (Nature, IF ~7.9) |
| **研究类型** | RCT |

| **干预** | VR 沉浸式放松 vs 引导意象 (TAU) |
| **效应量** | 两组均显著改善状态焦虑 (STAI-X1)、压力、消极情感、放松、积极情感、幸福感 (p<0.001)；VR 与 GI 无显著组间差异 |
| **可行性** | 低退出率、高满意度、高沉浸感、极低晕动症 (M=0.25±0.91) |

| **验证等级** | V2 (多来源交叉：OpenAlex 元数据 + 全文摘要) |
| **置信度** | 🟢 高 — Nature 子刊 RCT |

**产品行动建议**：
- ✅ **技术路线佐证**：VR/沉浸式技术在高龄（98岁）严重精神疾病人群仍可行 → 阿宝的语音对话式干预在可行性上更低门槛，有充足空间
- ✅ **评估指标参考**：STAI-X1 + PANAS + VAS (stress/relaxation/wellbeing) 可作为 Pilot 评估参考
- ⚠️ **注意**：两组无显著差异提示"参与本身可能是主要治疗因素"，支持我们"对话即干预"的叙事疗法路线

### 2. [arXiv] ASR 质量对 AD 语音检测影响的基准研究 ⭐⭐⭐⭐

| 字段 | 值 |
|------|-----|
| **标题** | Impact of automatic speech recognition quality on Alzheimer's disease detection from spontaneous speech: a reproducible benchmark study with lexical modeling and statistical validation |
| **日期** | 2026-03-18 |
| **来源** | arXiv (预印本) |
| **数据集** | ADReSSo 2021 诊断数据集 |
| **方法** | Whisper ASR 转录 → TF-IDF → Logistic Regression / Linear SVM，5×5 分层交叉验证 |
| **核心结果** | Whisper-small > Whisper-base，balanced accuracy > 0.7850 (Linear SVM)。统计配对检验确认差异显著 |
| **关键发现** | (1) 转录质量 > 分类器复杂度对性能的贡献 (2) 认知正常者产出更精确的对象/场景描述；AD 语言特征为模糊性、话语标记、犹豫模式增多 |
| **验证等级** | V2 (OpenAlex + 全文摘要交叉) |
| **置信度** | 🟡 中高 — arXiv 预印本，但方法学严谨（可复现 benchmark pipeline）|

**产品行动建议**：
- ✅ **直接影响 L0 评分器**：我们的 narrative-scorer 应优先保证 ASR 转录质量。Whisper-small 是当前性价比最优选择
- ✅ **叙事特征提取启发**：
  - "语义精确度" → 可增加到 L0 叙事评分维度
  - "犹豫模式"/"话语标记"密度 → 已有研究支持作为认知标记
  - "对象/场景描述丰富度" → 与上周 LLM 叙事评分研究（中心/外围信息）互补
- 📌 **Narrative Scorer v0.6 建议**：集成 ASR 质量校验层，在评分前先评估转录可靠性

## 📈 趋势总结

### 本周证据格局

1. **VR/沉浸式干预持续升温**：2 篇新研究（RCT + 定性），覆盖老年精神疾病和文化特异性记忆。但"VR vs 对照"常常组间无差异 → **"参与本身即治疗"** 假说得到加强
2. **ASR/NLP 用于认知检测进入工程化阶段**：不再只是 proof-of-concept，开始关注"哪个 ASR 模型更好""哪些词汇特征有区分力"
3. **被动传感 + 自适应干预**：Meta-Analysis 确认对认知障碍人群效果最大，但异质性高
4. **AI × 心理健康综述增多**：24 篇 RCT 的系统综述是迄今该方向最大规模汇总
5. **中国研究团队活跃度上升**：北大六院数字表型方案发表，中国语境证据积累中
