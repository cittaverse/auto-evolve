### 1. [BMC Geriatrics] 数字健康干预 SCD/MCI 系统综述 + Meta-Analysis ⭐⭐⭐⭐ 🆕

| 字段 | 值 |
|------|-----|
| **标题** | Digital health interventions for older adults with subjective cognitive decline or mild cognitive impairment: a systematic review and meta-analysis of randomized controlled trials |
| **DOI** | 10.1186/s12877-026-07341-w |
| **日期** | 2026-03-21 |
| **期刊** | BMC Geriatrics (IF ~4.1) |
| **研究类型** | 系统综述 + Meta-Analysis (PRISMA) |
| **纳入研究** | 15 篇 RCT |
| **总样本** | N=829 |
| **数据库** | 5 个数据库，截至 2025-08 |
| **核心效应量** | 认知功能: **SMD=0.387** (95% CI: 0.052–0.723, I²=66%) |
| **次要结果** | 功能能力: **MD=1.364** (95% CI: 0.937–1.790, I²=0.0%) ✅ 显著 |
| **无显著效果** | 情绪状态: SMD=-0.192 (95% CI: -0.405–0.021, I²=0.0%) ❌ |
| **无显著效果** | 生活质量: MD=-0.660 (95% CI: -2.730–1.409, I²=0.0%) ❌ |
| **验证等级** | V2 (OpenAlex 元数据 + 摘要完整提取) |
| **置信度** | 🟢 高 — BMC Geriatrics, PRISMA, 15 RCTs, 随机效应模型 |

**深度解读**：
- **效应量解析**: SMD=0.387 属于 **小到中等效应**（Cohen's d 标准），与上周被动传感 MA 的 SMD=3.22 形成鲜明对比。0.387 更接近临床真实预期
- **异质性**: I²=66% 表明研究间存在 **中等至较高异质性**，可能来自干预类型、频率、时长差异
- **关键分歧**: 认知功能改善显著，但 **情绪和生活质量无显著效果** — 这与上周 AI 正向精神科综述（24 RCTs 显示抑郁/焦虑改善）矛盾
- **矛盾解释**: 可能因为 (1) 本 MA 仅纳入 SCD/MCI 人群而非临床精神疾病人群; (2) 数字干预侧重认知训练而非情感支持; (3) 情绪量表在 SCD/MCI 人群天花板效应
- **局限性**: 作者明确指出"证据方法学质量有限"，需谨慎解读

**产品行动建议**：
- ✅ **核心参考**: SMD=0.387 是我们 Pilot 的 **效应量基准** — 如果阿宝的叙事干预能达到或超过此值，即为临床有意义
- ✅ **功能能力改善**: MD=1.364 功能能力改善 → 提示数字干预不仅改善测验分数，还改善日常功能 → 阿宝可设计"功能性叙事任务"（如叙述日常活动）
- ⚠️ **情绪无显著效果**: 阿宝的核心价值主张是"情感陪伴 + 回忆疗法" — 如果纯数字干预对情绪无效，需要论证阿宝的 **叙事+情感双通道** 与纯认知训练的本质区别
- 📌 **Pilot 设计影响**: 主要结局指标应同时覆盖认知（如 MoCA）和情绪（如 GDS/STAI），以检验"叙事疗法是否能突破纯数字干预的情绪天花板"
- 📌 **论文引用**: 必须在 Related Work 中引用此 MA 并讨论与我们方法的区别

### 2. [The Lancet] 口服 Semaglutide 治疗早期 AD — evoke/evoke+ Phase 3 ⭐⭐⭐ 🆕

| 字段 | 值 |
|------|-----|
| **标题** | Efficacy and safety of oral semaglutide 14 mg (flexible dose) in early-stage symptomatic Alzheimer's disease (evoke and evoke+): two phase 3, randomised, placebo-controlled trials |
| **作者** | Cummings JR, Atri A, Sano M, Zetterberg H, Scheltens P et al. |
| **DOI** | 10.1016/S0140-6736(26)00459-9 |
| **日期** | 2026-03-01 (本周新索引) |
| **期刊** | The Lancet (IF ~168) |
| **研究类型** | 2 项 Phase 3 RCT |
| **设计** | 多中心、随机、双盲、安慰剂对照 |
| **场景** | 566 个站点，40 个国家 |
| **人群** | 55-85 岁，淀粉样蛋白确认的 AD，MCI 或轻度痴呆 |
| **干预** | 口服 Semaglutide 14mg/日 (弹性剂量) |
| **赞助** | Novo Nordisk |
| **验证等级** | V1 (OpenAlex 索引，摘要不完整) |
| **置信度** | 🟢 高 — The Lancet, Phase 3, 大规模 |

**深度解读**：
- 这是 **GLP-1 受体激动剂治疗 AD** 的里程碑试验。Semaglutide (Ozempic/Wegovy 的活性成分) 正在从代谢领域向神经退行性疾病扩展
- 无法从当前摘要提取具体效应量和主要终点结果 (OpenAlex 索引不完整)
- 即使非我们直接产品赛道，此试验对 AD 市场格局影响深远：如果药物治疗取得突破，非药物干预（包括回忆疗法）的定位将从"替代方案"转向"联合方案"

**产品行动建议**：
- ✅ **市场定位调整**: 如 semaglutide 获批用于 AD → 阿宝应定位为"药物+叙事疗法联合方案"的非药物组件，而非独立替代
- 📌 **待追踪**: 需获取全文以了解具体疗效数据。关注 FDA/EMA 后续审批进展
- 📌 **竞品分析**: Novo Nordisk 进入 AD 领域意味着大药企资本将涌入，数字疗法需找到与药物互补而非竞争的生态位

### 3. [GeroScience] tDCS 增强言语流畅性 — 预注册 RCT ⭐⭐ 🆕

| 字段 | 值 |
|------|-----|
| **标题** | Baseline cognition and stimulation site shape the effects of tDCS on verbal fluency in older and younger adults |
| **作者** | Yücel A, Martin AK |
| **DOI** | 10.1007/s11357-026-02104-2 |
| **日期** | 2026-03-23 |
| **期刊** | GeroScience (IF ~5.3) |
| **研究类型** | 预注册、双盲、假刺激对照 RCT |
| **样本** | N=144 (72 年轻 + 72 老年) |
| **干预** | 阳极 tDCS → 左侧 IFG 或 左侧 TPJ |
| **核心结果** | 左 TPJ 阳极刺激显著改善音位流畅性 (F(1,71)=4.49, p=.038); 语义流畅性和左 IFG 刺激无效 |
| **调节变量** | 流体智力 (而非年龄) 调节刺激反应: 流体智力较低者获益更大 (F(1,70)=7.80, p=.007) |
| **验证等级** | V2 (OpenAlex + 完整摘要) |
| **置信度** | 🟢 高 — 预注册、双盲、假刺激对照、N=144 |

**深度解读**：
- **"流体智力低者获益更大"** 这一发现高度相关：认知功能较弱的个体从干预中获益更多 → 与上周被动传感 MA 的"认知障碍人群获益最大"一致
- tDCS 本身不是我们的技术路线，但其揭示的 **"言语流畅性的神经基础是可调节的"** 对阿宝有间接价值
- 左 TPJ (颞顶联合区) 是 **叙事理解和心智化** 的关键脑区 → 增强 TPJ 可能同时增强叙事能力

**产品行动建议**：
- ✅ **评估指标参考**: 音位/语义流畅性测试可作为阿宝 Pilot 的 **次要认知指标** — 简单、标准化、对变化敏感
- 📌 **远期探索**: tDCS+叙事疗法联合干预是一个有理论基础的方向（TPJ 同时服务于流畅性和叙事心智化）
- 📌 **个体化干预**: "认知基线调节干预效果"强化了阿宝需要 **根据用户认知水平调整对话难度** 的设计理念

## 📈 趋势分析

### 本周证据格局更新

1. **数字干预对 MCI/SCD 认知有效的证据进一步固化**
   - BMC Geriatrics MA (N=829, 15 RCTs) 给出 SMD=0.387 → 这是迄今 **最大规模的 SCD/MCI 数字干预 MA**
   - 但 **情绪/QoL 无显著效果** 是一个重要信号 → 需要区分"认知训练型"和"情感叙事型"数字干预

2. **药物赛道 AD 突破在即**
   - Semaglutide Phase 3 (Lancet) 标志着 GLP-1 RA 从代谢向神经退行性扩展
   - 如果药物获批，非药物干预的生态位将重新定义

3. **"低认知基线者获益更大"成为跨模态一致发现**
   - 被动传感 MA (上周): 认知障碍人群效果最大
   - tDCS RCT (本周): 流体智力低者获益更大
   - → 阿宝的目标人群（轻度认知下降的老年人）恰好是获益最大的群体

4. **上周全部 9 条结论保持有效**，无新证据推翻

### 关键矛盾与待解决问题

| 矛盾 | 来源 | 可能解释 | 待验证 |
|------|------|---------|--------|
| 数字干预改善认知但不改善情绪 (BMC) vs AI chatbot 改善抑郁/焦虑 (Healthcare SR) | 两篇本周/上周 | 人群差异 (SCD/MCI vs 临床精神疾病) + 干预类型差异 (认知训练 vs 对话陪伴) | 阿宝 Pilot 需同时测量认知+情绪以验证叙事疗法是否能两者兼顾 |
