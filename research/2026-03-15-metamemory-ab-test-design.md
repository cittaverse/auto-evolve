# 元记忆 A/B 测试设计方案

**日期**: 2026-03-15  
**阶段**: 元记忆整合研究 - 阶段 5 (A/B 测试设计)  
**截止**: 2026-03-20  
**状态**: 草稿完成

---

## 研究问题

**核心假设**: 在叙事引导中加入元认知提示 (metacognitive prompts) 可提升:
1. 叙事细节丰富度 (内部细节 + 外部细节)
2. 记忆信心评分 (1-5 分)
3. 叙事连贯性评分

---

## 实验设计

### 1. 被试分组

| 组别 | N | 引导策略 | 预期效果 |
|------|---|----------|----------|
| **A 组 (控制组)** | 30 | 标准叙事引导 (5 维度) | 基线表现 |
| **B 组 (实验组)** | 30 | 标准引导 + 元认知提示 | 细节↑20-30%, 信心↑1.5-2.0 分 |

**纳入标准**:
- 年龄 60-80 岁
- MoCA 评分 18-25 (MCI 范围)
- 能独立完成 30 分钟语音会话
- 知情同意书签署

**排除标准**:
- 重度听力障碍 (影响语音交互)
- 严重精神疾病史
- 过去 3 月内参与其他认知干预研究

### 2. 干预方案

**A 组 (控制组) - 标准引导**:
```
1. 感官细节: "您当时看到了什么？听到了什么？闻到了什么？"
2. 情境细节: "那是在哪里？什么时候？周围有谁？"
3. 情感深度: "您当时感觉如何？这件事让您有什么情绪？"
4. 事件分段: "这件事是怎么开始的？中间发生了什么？最后怎样结束？"
5. 连贯性: "这件事和您之前的经历有什么联系？"
```

**B 组 (实验组) - 标准引导 + 元认知提示**:
```
【在标准引导基础上，每 3-5 分钟插入元认知提示】

来源监控 (Source Monitoring):
- "您是如何记住这件事的？是自己经历的，还是别人告诉您的？"
- "您当时是直接看到的，还是后来听说的？"

信心评估 (Confidence Rating):
- "您对这件事的记忆有多确定？(1-5 分)"
- "有哪些细节您记得特别清楚？哪些不太确定？"

策略提示 (Strategy Prompting):
- "试着从另一个角度回忆这件事，会有什么不同？"
- "如果让您用三个词总结这件事，会是哪三个词？"

意义反思 (Meaning Reflection):
- "这件事对您来说意味着什么？"
- "这件事如何影响了您后来的生活？"
```

### 3. 测量指标

| 维度 | 指标 | 测量工具 | 时点 |
|------|------|----------|------|
| **主要结局** | 叙事细节总数 | L0 六维评估系统 | 基线、8 周 |
| | 内部细节数 | L0 六维评估系统 | 基线、8 周 |
| | 外部细节数 | L0 六维评估系统 | 基线、8 周 |
| **次要结局** | 记忆信心评分 | 5 点 Likert 量表 | 每次会话后 |
| | 叙事连贯性 | L0 连贯性维度 | 基线、8 周 |
| | 认知功能 | MoCA | 基线、8 周、3 月随访 |
| | 抑郁症状 | GDS-15 | 基线、8 周、3 月随访 |
| | 生活质量 | QOL-AD | 基线、8 周 |
| **过程指标** | 会话完成率 | % | 每次会话 |
| | 平均会话时长 | 分钟 | 每次会话 |
| | 元认知提示响应率 | % | B 组专用 |

### 4. 样本量计算

**效应量假设** (基于 JMIR mHealth 2026-01):
- 叙事细节：预期效应量 d = 0.5 (中等效应)
- 记忆信心：预期效应量 d = 0.6 (中等效应)

**计算参数**:
- α = 0.05 (双尾)
- 1-β = 0.80 (统计功效)
- d = 0.5 (保守估计)

**结果**:
- 每组需要 N = 64 人 (G*Power 计算)
- 考虑 20% 脱落率 → 每组 N = 80 人
- **总样本量**: N = 160 人

**Pilot 阶段调整** (可行性测试):
- 每组 N = 30 人 (总 N = 60)
- 目的：验证干预可行性、依从性、测量工具信度
- 正式 RCT 待 Pilot 完成后启动

### 5. 随机化与盲法

**随机化方法**:
- 计算机生成随机序列 (区组随机化，区组大小=4)
- 1:1 分配至 A 组或 B 组
- 分配隐藏：密封不透明信封 (SNOSE)

**盲法**:
- 被试：单盲 (不知分组目的)
- 评估者：双盲 (不知分组)
- 干预实施者：无法设盲 (需执行不同引导策略)

### 6. 统计分析计划

**主要分析**:
- 意向性分析 (ITT): 所有随机化被试纳入分析
- 组间比较：独立样本 t 检验 (连续变量) / 卡方检验 (分类变量)
- 纵向变化：线性混合模型 (LMM)，控制基线值

**次要分析**:
- 符合方案集分析 (PP): 完成≥80% 干预 sessions
- 亚组分析：按 MoCA 基线分层 (18-21 vs 22-25)
- 中介分析：元认知提示响应率是否中介效果

**软件**:
- R 4.0+ (lme4, emmeans, mediation 包)
- α = 0.05 (双尾)

### 7. 伦理考量

**风险**:
- 回忆负面事件可能引发情绪波动
- 对策：会话后情绪评估 + 转介心理支持

**获益**:
- 认知功能潜在改善
- 情绪状态潜在改善
- 生命意义感提升

**知情同意**:
- 书面知情同意书
- 被试可随时退出
- 数据匿名化处理

**伦理审批**:
- 牵头单位伦理委员会审批
- 试验注册：ChiCTR (中国临床试验注册中心)

### 8. 数据质量控制

**数据采集质控**:
| 风险 | 控制措施 | 负责人 |
|------|----------|--------|
| 录音质量差 | 统一设备 (iPhone 内置麦克风) + 安静环境要求 | 合作单位 |
| 转写错误 | ASR 双引擎校验 (Azure + 讯飞) + 人工抽检 10% | Hulk |
| 脱落率高 | 每周跟进 + 交通补贴 + 灵活预约时间 | 合作单位 |
| 干预依从性低 | 会话提醒 + 完成率监控 + 低于 80% 电话随访 | 合作单位 |

**评估者一致性**:
- L0 评分系统：自动化评分 (消除人为主观性)
- 人工抽检：随机抽取 10% 样本，2 名评估者独立评分
- 一致性指标：Cohen's Kappa ≥ 0.80

**数据完整性**:
- 缺失数据处理：多重插补法 (Multiple Imputation)
- 离群值检测：箱线图法 + 临床合理性判断
- 数据锁定：分析前由独立统计师审核

**盲态保持**:
- 评估者不知分组
- 数据分析采用编码组别 (A/B→1/2)
- 揭盲仅在数据库锁定后进行

## 附录：元认知提示 YAML 配置 (草案)

```yaml
metamemory_prompts:
  source_monitoring:
    - "您是如何记住这件事的？是自己经历的，还是别人告诉您的？"
    - "您当时是直接看到的，还是后来听说的？"
    - "这件事是您亲身参与的，还是旁观的？"
    
  confidence_rating:
    - "您对这件事的记忆有多确定？请用 1-5 分评分 (1=完全不确定，5=非常确定)"
    - "有哪些细节您记得特别清楚？哪些不太确定？"
    - "如果让您向别人讲述这件事，您有多大把握？"
    
  strategy_prompting:
    - "试着从另一个角度回忆这件事，会有什么不同？"
    - "如果让您用三个词总结这件事，会是哪三个词？"
    - "闭上眼睛，想象自己回到当时的情景，您首先注意到什么？"
    
  meaning_reflection:
    - "这件事对您来说意味着什么？"
    - "这件事如何影响了您后来的生活？"
    - "现在回想起来，您对这件事有什么新的理解？"

delivery_rules:
  frequency: "每 3-5 分钟插入 1 条元认知提示"
  selection_strategy: "根据被试响应动态选择 (信心低→信心评估; 细节少→策略提示)"
  max_prompts_per_session: 8
```
