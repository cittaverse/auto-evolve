#### 2.2 变量控制增强 (四层 → 四层 + 抗堆砌专项)

**新增第五层控制**: 抗堆砌专项控制

| 控制层级 | 控制对象 | 主要方法 | 负责人 | 容差标准 | 监测频率 |
|----------|----------|----------|--------|----------|----------|
| **抗堆砌层** | 关键词堆砌检测 | L0 密度检测 + L1 语义一致性 | 系统自动 | 堆砌率<2% | 实时 |

**堆砌类型定义**:

| 堆砌类型 | 检测信号 | 阈值 | 处理策略 |
|----------|----------|------|----------|
| **C1 感官堆砌** | 感官词密度 >3 SD | >6 词/100 字 | L1 仲裁 + 评分打折 |
| **C2 实体堆砌** | 外部实体密度 >3 SD | >8 实体/100 字 | L1 仲裁 + 评分打折 |
| **C4 情感堆砌** | 情感词密度 >3 SD + 效价不匹配 | >5 词/100 字 | L1 仲裁 + 评分打折 |
| **混合堆砌** | 多维度同时异常 | ≥2 维度异常 | L1 仲裁 + 人工复核 |

**处理策略分级**:

| 严重程度 | 处理方式 | 评分调整 |
|----------|----------|----------|
| **无堆砌** | 正常评分 | 无调整 |
| **轻度 (1 维度)** | L1 仲裁 + 警告标记 | 综合分 × 0.95 |
| **中度 (2 维度)** | L1 仲裁 + 人工复核 | 综合分 × 0.85 |
| **重度 (3+ 维度)** | 人工复核 + 可能排除 | 综合分 × 0.70 或排除 |

**检测算法伪代码**:

```python
def detect_gaming(text, scores):
    gaming_signals = {
        'sensory_density': count_sensory_words(text) / len(text) * 100,
        'entity_density': count_external_entities(text) / len(text) * 100,
        'emotion_density': count_emotion_words(text) / len(text) * 100,
        'emotion_event_mismatch': check_emotion_event_consistency(text, scores)
    }
    
    thresholds = {
        'sensory_density': 6.0,
        'entity_density': 8.0,
        'emotion_density': 5.0,
        'emotion_event_mismatch': True
    }
    
    detected = [k for k, v in gaming_signals.items() 
                if v > thresholds[k] or (k == 'emotion_event_mismatch' and v)]
    
    return {
        'is_gaming': len(detected) >= 1,
        'gaming_types': detected,
        'severity': len(detected)
    }
```
