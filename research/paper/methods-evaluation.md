# 评估方案 (Evaluation)

## 评估指标

本研究从三个层面评估系统效果：叙事质量、用户体验、心理健康结果。

### 3.3.1 叙事质量指标（主要终点）

| 指标 | 定义 | 测量方式 |
|------|------|----------|
| L0 六维度分数 | 各维度 0-5 分 | LLM 自动评分 |
| 叙事长度 | 总字数 | 文本统计 |
| 事件数量 | 提取的事件节点数 | 记忆图谱 |
| 时间跨度 | 覆盖的人生年限 | 时间标注分析 |
| 主题多样性 | 涉及的主题类别数 | 主题模型聚类 |

### 3.3.2 用户体验指标（次要终点）

| 指标 | 定义 | 测量方式 |
|------|------|----------|
| SUS | 系统可用性量表（10 题） | 问卷（0-100） |
| NPS | 净推荐值 | "您会推荐给朋友吗？"（-10 到 10） |
| 完成率 | 完成预设 session 数的用户比例 | 行为日志 |
| 平均 session 时长 | 单次对话持续时间 | 时间戳差值 |
| 流失率 | 中途放弃的用户比例 | 行为日志 |

### 3.3.3 心理健康指标（探索性终点）

| 指标 | 量表 | 测量时点 |
|------|------|----------|
| 抑郁症状 | GDS-15（老年抑郁量表） | 基线、4 周、8 周 |
| 生活满意度 | SWLS（5 题） | 基线、4 周、8 周 |
| 孤独感 | UCLA Loneliness Scale（3 题简版） | 基线、4 周、8 周 |
| 自我连续性 | Self-Continuity Scale（4 题） | 基线、4 周、8 周 |

## 基线对比

为验证本系统的有效性，我们设计了三组对照：

### 3.3.4 对照组设计

| 组别 | 干预方式 | 样本量 | 假设 |
|------|----------|--------|------|
| A（实验组） | 完整系统（引导 + 评分 + 反馈） | n=80 | 最优效果 |
| B（引导仅组） | 仅 LLM 引导，无评分反馈 | n=80 | 验证评分反馈的增量价值 |
| C（自由书写组） | 无引导，自由输入 | n=80 | 验证引导的价值 |
| D（等待对照组） | 无干预，8 周后交叉 | n=40 | 控制时间效应 |

### 3.3.5 统计检验

主要分析采用**意向性分析（ITT）**原则，缺失数据使用多重插补法处理。

- **组间比较**：ANCOVA，以基线分数为协变量
- **时间效应**：重复测量 ANOVA（基线、4 周、8 周）
- **效应量**：Cohen's d（组间）、η²（时间）
- **显著性水平**：α = 0.05，双侧检验
- **多重比较校正**：Bonferroni 校正

样本量计算基于 pilot study 数据：预期 GDS-15 组间差异 d = 0.5，α = 0.05，power = 0.8，每组需 64 人。考虑 20% 流失率，每组目标 80 人。

## 消融实验

为验证系统各组件的贡献，我们进行了以下消融实验：

### 3.3.6 组件消融

| 实验 | 移除组件 | 预期影响 | 验证指标 |
|------|----------|----------|----------|
| E1 | 移除记忆图谱 | 叙事连贯性下降 | 时间连贯性分数 |
| E2 | 移除策略选择 | 引导多样性下降 | 主题多样性分数 |
| E3 | 移除评分反馈 | 用户参与度下降 | session 完成率 |
| E4 | 移除五感提示 | 感官细节减少 | 感官细节分数 |
| E5 | 移除社会联结提示 | 社会提及减少 | 社会联结分数 |

### 3.3.7 提示工程消融

为验证 prompt 设计的有效性，我们对比了四种 prompt 策略：

| 策略 | 描述 | 特点 |
|------|------|------|
| P1（零样本） | 直接指令，无示例 | 最简 |
| P2（少样本） | 包含 3 个示例 | 标准 |
| P3（思维链） | 要求逐步推理 | 增强推理 |
| P4（结构化输出） | 强制 JSON 格式 | 便于解析 |

评估指标为评分一致性（与人类评分者 ICC）与解析成功率（JSON 有效比例）。

### 3.3.8 模型选择消融

对比四种 LLM 后端的性能与成本：

| 模型 | 提供商 | 参数量 | 单次评分成本 | 评分一致性 (ICC) |
|------|--------|--------|--------------|------------------|
| Qwen-Plus | 阿里云 | ~100B | ¥0.004 | 0.78 |
| GPT-4o-mini | OpenAI | ~10B | $0.00015 | 0.75 |
| Claude-3-Haiku | Anthropic | ~10B | $0.00025 | 0.76 |
| GLM-4-Flash | 智谱 | ~10B | ¥0.001 | 0.72 |

最终选择 Qwen-Plus 基于综合考量：中文能力最优、成本可控、数据主权（国内部署）。

## 伦理考量

本研究已通过浙江大学伦理委员会审批（批准号：2025-XXX）。关键伦理措施包括：

1. **知情同意**：用户签署电子知情同意书，明确数据用途与退出权利
2. **隐私保护**：所有数据匿名化处理，移除姓名、地址、电话等直接标识符
3. **风险监测**：设置自杀/自伤关键词检测，触发时转接人工干预
4. **数据最小化**：仅收集研究必需的数据，8 周后用户可选择删除全部数据
5. **公平性**：招募时确保性别、年龄、教育程度的多样性，避免样本偏差
