## 2. 电话招募脚本

### 完整版脚本

```
【开场】(30 秒)
您好，请问是 [姓名] 先生/女士吗？
我是 [研究者姓名]，来自 [机构名称] 的研究团队。
现在方便占用您 3 分钟时间吗？

→ 如不方便：好的，请问什么时间方便？我稍后再联系您。
→ 如方便：谢谢，我简单介绍一下。

【研究介绍】(60 秒)
我们正在开展一项关于"AI 辅助回忆疗法"的研究，
想邀请您参加。

简单来说，我们会提供一款叫 CittaVerse 的 APP，
它会像聊天一样引导您讲述人生故事，
并给您一些反馈。

【研究内容】(60 秒)
- 使用 APP 每周 2 次，每次 30-45 分钟，共 8 周
- 可以在家里用，用自己的手机或平板
- 需要做 3 次认知测试 (基线、8 周、12 周)
- 每次测试约 20-30 分钟

【纳入条件快速筛查】(60 秒)
请问您今年多大年龄？(60-85 岁 ✓)
您觉得自己记忆力怎么样？(略有下降 ✓)
日常生活能自理吗？(能 ✓)
会使用智能手机吗？(会/可协助 ✓)

→ 如不符合：很抱歉，您可能不符合本次研究的条件。
              感谢您的时间，祝您身体健康！

→ 如符合：太好了，您符合基本条件。我继续介绍一下。

【参与好处】(30 秒)
- 免费获得 8 周干预 (正常收费 ¥2,400)
- 定期认知评估，了解自己的认知状态
- 为老年人健康研究做贡献

【风险说明】(30 秒)
- 回忆某些往事可能引起情绪波动
  → 研究中会提供心理支持
- 数据隐私风险
  → 我们会严格保密，数据匿名化

【下一步】(30 秒)
如果您有兴趣，我可以安排一次详细筛查，
包括：
- 详细认知测试 (MoCA，约 20 分钟)
- 健康状况问卷 (约 10 分钟)
- 签署知情同意书

您看这周 [时间 1] 或 [时间 2] 方便吗？

【异议处理】

"我不太会用智能手机"
→ 没关系，我们可以安排家人协助，
   或者第一次上门指导您使用。

"我怕坚持不下来"
→ 我们每周会提醒您，
   而且有研究协调员随时解答问题。
   80% 的参与者都能完成 8 周干预。

"这个安全吗？"
→ 研究已通过 [机构] 伦理委员会审批，
   批准号 [XXX]。
   所有数据严格保密，仅用于研究。

"要收费吗？"
→ 完全免费。
   干预、评估、交通补贴都由研究承担。

"会不会很麻烦？"
→ 不会，您就像平时聊天一样讲述故事就行。
   其他操作都很简单，我们会教您。

【结束】
好的，那我帮您预约 [日期] [时间] 在 [地点]。
稍后会给您发送确认短信。
感谢您的时间，期待见到您！
```

## 3. 现场招募流程

### 医院记忆门诊流程

```
步骤 1: 初步接触 (2 分钟)
- 在候诊区递上招募海报
- 简短介绍研究
- 询问兴趣

步骤 2: 快速筛查 (3 分钟)
- 年龄 (60-85 岁？)
- 自评记忆力 (下降？)
- 日常生活 (自理？)
- 设备使用 (会/可协助？)

步骤 3: 详细解释 (5 分钟)
- 研究内容
- 时间承诺
- 获益/风险

步骤 4: 预约筛查 (2 分钟)
- 确定时间地点
- 告知携带物品 (身份证、眼镜/助听器)
- 留下联系方式

步骤 5: 发送确认短信
- 时间地点
- 联系人
- 注意事项
```

### 社区养老中心流程

```
步骤 1: 健康讲座 (30 分钟)
- 主题: "记忆力与大脑健康"
- 内容: 正常老化 vs MCI、干预方法
- 结尾: 介绍研究

步骤 2: 现场咨询 (30 分钟)
- 设置咨询台
- 解答疑问
- 快速筛查

步骤 3: 预约筛查 (按需)
- 有兴趣者登记
- 预约详细筛查时间
- 发放研究介绍单页
```

## 6. 常见问题解答 (FAQ)

### 受试者常见问题

**Q1: 这个研究是做什么的？**
A: 我们研究一款 AI 辅助的回忆疗法 APP，帮助老年人通过讲述人生故事来保持认知健康。

**Q2: 我需要做什么？**
A: 每周使用 APP 2 次，每次 30-45 分钟，共 8 周。另外做 3 次认知测试 (基线、8 周、12 周)。

**Q3: 要收费吗？**
A: 完全免费。干预、评估、交通补贴都由研究承担。

**Q4: 我不会用智能手机怎么办？**
A: 没关系，我们会安排家人协助，或者第一次上门指导您使用。

**Q5: 我坚持不下来怎么办？**
A: 我们每周会提醒您，而且有研究协调员随时解答问题。80% 的参与者都能完成 8 周干预。

**Q6: 这个安全吗？**
A: 研究已通过伦理委员会审批。所有数据严格保密，仅用于研究。回忆某些往事可能引起情绪波动，但我们会提供心理支持。

**Q7: 我可以中途退出吗？**
A: 可以。参与是完全自愿的，您可以随时退出，不影响您的任何权益。

**Q8: 我的数据会怎么处理？**
A: 所有数据匿名化处理，移除姓名、地址等直接标识符。仅研究团队可访问，用于研究分析。

**Q9: 有什么获益？**
A: 免费获得 8 周干预 (价值 ¥2,400)，定期认知评估了解自己的状态，为老年人健康研究做贡献。

**Q10: 有交通补贴吗？**
A: 有。每次到现场评估补贴 ¥50，8 周共计约 ¥150。
