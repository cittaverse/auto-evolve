# 评分算法 (Scoring Algorithm)

## L0 六维度详解

本研究的评分体系建立在自传体记忆研究与叙事心理学的理论基础之上。六维度的选择基于以下原则：(1) 覆盖叙事质量的核心方面；(2) 可操作化为 LLM 可评估的指标；(3) 与心理健康结果存在已知关联。

### 3.2.1 时间连贯性 (Temporal Coherence)

**定义**：叙事中时间线索的清晰度与事件排序的逻辑性。高时间连贯性的叙事能够明确标注事件发生的时间顺序，并使用恰当的时间连接词（如"之后"、"在此之前"、"与此同时"）。

**评分标准**：
| 分数 | 描述 | 示例特征 |
|------|------|----------|
| 0 | 无时间信息 | 完全缺失时间词 |
| 1 | 模糊时间 | "小时候"、"那时候" |
| 2 | 相对时间 | "上学后"、"结婚前" |
| 3 | 年代标注 | "80 年代"、"1990 年" |
| 4 | 精确时间 | "1985 年 6 月" |
| 5 | 时间网络 | 多事件时间关系清晰 |

**Prompt 示例**：
```
请评估以下文本的时间连贯性（0-5 分）：
- 是否包含明确的时间标注？
- 事件之间的时间关系是否清晰？
- 是否使用时间连接词？
输出 JSON: {"score": int, "reason": "string"}
```

### 3.2.2 情感效价 (Emotional Valence)

**定义**：叙事中表达的情感倾向。基于积极心理学研究，平衡的情感表达（既有积极也有消极）与更好的心理健康结果相关。

**评分标准**：
- **效价方向**：积极（+）、中性（0）、消极（-）
- **效价强度**：弱（1）、中（2）、强（3）
- **情感复杂度**：单一情感（1）、混合情感（2）

综合分数 = 基础分（2 分）+ 情感复杂度（0-2 分）+ 效价平衡bonus（1 分，若同时包含积极与消极）

**理论依据**：Labov & Waletzky (1967) 的叙事评估理论强调情感表达是叙事功能的核心；积极心理学研究表明，能够整合消极经历并赋予意义的个体具有更高的心理韧性。

### 3.2.3 感官细节 (Sensory Details)

**定义**：叙事中包含的感官体验描述，涵盖视觉、听觉、嗅觉、味觉、触觉五类。

**评分标准**：
| 分数 | 感官类别数量 | 示例 |
|------|--------------|------|
| 0 | 0 | 纯抽象描述 |
| 1 | 1 | "天很蓝" |
| 2 | 2 | "天很蓝，鸟在叫" |
| 3 | 3 | 加入嗅觉/触觉 |
| 4 | 4 | 四类感官 |
| 5 | 5 | 五感俱全 |

**评分算法**：
```python
def score_sensory_details(text):
    senses = {
        'visual': ['看见', '蓝色', '明亮', '黑暗', ...],
        'auditory': ['听到', '声音', '吵闹', '安静', ...],
        'olfactory': ['闻到', '香味', '臭味', ...],
        'gustatory': ['尝到', '甜', '苦', ...],
        'tactile': ['摸到', '温暖', '冰冷', '粗糙', ...]
    }
    detected = set()
    for sense, keywords in senses.items():
        if any(kw in text for kw in keywords):
            detected.add(sense)
    return min(5, len(detected))
```

### 3.2.4 社会联结 (Social Connection)

**定义**：叙事中涉及的社会关系与他人提及。社会联结是晚年心理健康的保护性因素，高质量的叙事通常包含丰富的社会网络描述。

**评分维度**：
- **关系多样性**：提及的关系类型数量（家人/朋友/同事/邻居等）
- **互动深度**：是否描述具体的互动行为（对话/合作/冲突/支持）
- **情感纽带**：是否表达对关系的情感评价（感激/怀念/遗憾）

**综合分数** = 关系多样性（0-2 分）+ 互动深度（0-2 分）+ 情感纽带（0-1 分）

### 3.2.5 事件分段 (Event Segmentation)

**定义**：叙事中事件的边界清晰度。认知心理学研究表明，人类记忆以"事件"为基本单位存储，清晰的事件分段有助于记忆的提取与整合。

**评分标准**：
| 分数 | 描述 |
|------|------|
| 0 | 完全流水账，无事件边界 |
| 1 | 模糊分段，依靠段落划分 |
| 2 | 有时间/地点转换提示 |
| 3 | 有明确的事件起始与结束 |
| 4 | 事件之间有因果/主题联系 |
| 5 | 层次化结构（大事件包含小事件） |

### 3.2.6 自我反思 (Self-Reflection)

**定义**：叙事中对自身经历的意义建构与元认知评论。自我反思是叙事身份（narrative identity）形成的核心机制，与人生满意度显著相关。

**反思类型**：
- **因果反思**：解释事件发生的原因（"之所以...是因为..."）
- **成长反思**：描述经历带来的改变（"从那以后我学会了..."）
- **价值反思**：表达对经历的价值判断（"现在想来，那是..."）
- **整合反思**：将个别经历与人生主题联系（"这体现了我一贯的..."）

**评分**：每出现一种反思类型 +1 分，上限 5 分。

## 权重设计

六维度的权重分配基于两个原则：(1) 理论重要性；(2) 与结果变量的预测效度。我们采用**层次分析法（AHP）**确定初始权重，并通过 pilot study 数据进行经验校正。

### 初始权重（AHP）

邀请 3 位叙事心理学专家对六维度进行两两比较，构建判断矩阵，计算特征向量得到初始权重：

| 维度 | 权重 | 理论依据 |
|------|------|----------|
| 时间连贯性 | 0.15 | 叙事结构基础 |
| 情感效价 | 0.20 | 心理健康关联最强 |
| 感官细节 | 0.15 | 记忆生动性指标 |
| 社会联结 | 0.20 | 晚年福祉保护因子 |
| 事件分段 | 0.15 | 认知加工质量 |
| 自我反思 | 0.15 | 意义建构核心 |

### 经验校正

基于 200+ 用户的 pilot study 数据，我们以"用户满意度"与"继续讲述意愿"为因变量，进行回归分析，发现情感效价与社会联结的预测力最强。校正后权重：

| 维度 | 初始权重 | 校正权重 | 变化 |
|------|----------|----------|------|
| 时间连贯性 | 0.15 | 0.12 | -0.03 |
| 情感效价 | 0.20 | 0.25 | +0.05 |
| 感官细节 | 0.15 | 0.13 | -0.02 |
| 社会联结 | 0.20 | 0.25 | +0.05 |
| 事件分段 | 0.15 | 0.12 | -0.03 |
| 自我反思 | 0.15 | 0.13 | -0.02 |

### 聚合公式

单轮叙事质量分数：
$$Quality_{turn} = \sum_{i=1}^{6} w_i \times score_i$$

会话级分数（考虑时间衰减）：
$$Quality_{session} = \frac{\sum_{t=1}^{T} Quality_{turn,t} \times \lambda^{T-t}}{\sum_{t=1}^{T} \lambda^{T-t}}$$

其中 $\lambda = 0.95$ 为衰减因子，赋予近期叙事更高权重。

用户级分数（考虑主题多样性）：
$$Quality_{user} = Quality_{session\_avg} \times (1 + 0.1 \times topic\_diversity)$$

其中 $topic\_diversity$ 为不同主题 session 的数量（上限 5）。

## 评分一致性验证

为确保 LLM 评分的可靠性，我们进行了以下验证：

1. **评分者间信度**：随机抽取 100 段叙事，由 2 位人类评分者与 LLM 分别评分，计算 ICC（组内相关系数）。结果：LLM vs 人类平均 ICC = 0.78，达到可接受水平。

2. **重测信度**：对同一段叙事进行 5 次 LLM 评分（不同 seed），计算变异系数。结果：CV < 0.1，表明评分稳定。

3. **效标效度**：以 L0 六维度分数预测 UWES（Utrecht Work Engagement Scale）与 SWLS（Satisfaction With Life Scale），R² = 0.42，表明评分具有外部效度。
