| 结果对齐 | 高质量叙事 → B-A 级; 低质量 → F-D 级 |

### 1.2 待完成: Benchmark 数据集

**状态**: 🟡 设计中 (GEO #63 计划)

**设计**:
- 创建 10-20 段手工标注的中文叙事 (gold standard)
- 每段叙事由 2 名中文母语评估者独立标注 6 个维度
- 计算评估者间信度 (Cohen's κ 或 ICC)
- 计算自动评分与人工标注的相关性 (Pearson r)

**标注协议**:

| 维度 | 标注方式 | 评分范围 |
|------|----------|----------|
| 事件丰富度 | 数事件数 + 整体评分 | 0-100 |
| 时间连贯性 | 标记时间标记 + 整体评分 | 0-100 |
| 因果连贯性 | 标记因果连接 + 整体评分 | 0-100 |
| 情感深度 | 标记情感词 + 整体评分 | 0-100 |
| 自我认同 | 标记自我指称 + 整体评分 | 0-100 |
| 信息密度 | 标记中心/周边 + 比例 | 0-100 |

**目标指标**:
- 评估者间信度: κ > 0.75 (substantial agreement)
- 自动 vs 人工相关: r > 0.70

### 1.3 待完成: v0.5 vs v0.6 对比

## 实验 2: Pilot RCT (L2)

### 2.1 研究设计

**设计类型**: 多中心、随机、对照、单盲 Pilot RCT

| 参数 | 值 |
|------|---|
| **样本量** | N=60-80 (30-40/组) |
| **干预时长** | 8 周 (每周 2 次) |
| **对照** | 主动对照 (结构化认知训练) |
| **盲法** | 单盲 (评估者盲) |
| **随机化** | 1:1 区组随机 (block size=4) |
| **中心数** | 2-3 (杭州社区 + 记忆门诊) |

### 2.2 受试者

**纳入标准**:
- 年龄 60-85 岁
- MCI (MoCA 18-25)
- 普通话或当地方言
- 矫正后可正常交流
- 知情同意

**排除标准**:
- 重度痴呆 (MoCA <18)
- 严重精神疾病
- 近 3 月参加其他 RCT

### 2.3 干预方案

**干预组 (CittaVerse)**:
```
每次会话流程 (30-40 分钟):
1. AI 引导回忆特定主题 (15-20 分)
2. ASR 实时转写 + 六维评分
3. 元记忆策略提示 (基于评分反馈)
4. 人类专家 10% 抽检
5. 生成"人生故事书"章节
```

**对照组 (Active Control)**:
- 结构化认知训练 (记忆游戏 + 注意力任务)
- 相同频率和时长
- 不含叙事/回忆成分

### 2.4 结局指标

**主要终点**:
| 指标 | 工具 | 时点 |
|------|------|------|
| 认知功能变化 | MoCA | 基线、8 周 |

**次要终点**:
| 指标 | 工具 | 时点 |
|------|------|------|
| 记忆力 | 逻辑记忆测试 (LM) | 基线、8 周 |
| 抑郁症状 | GDS-15 | 基线、8 周、3 月随访 |
| 生活质量 | QOL-AD | 基线、8 周 |
| 生活满意度 | SWLS | 基线、8 周 |
| 行为症状 | NPI-Q | 基线、8 周 |

**探索性终点**:
| 指标 | 测量方式 |
|------|----------|
| 六维评分变化 | CittaVerse 自动评分 |
| 叙事细节数量 | 事件计数 |
| 会话完成率 | 系统记录 |
| 用户满意度 | 自定义问卷 + NPS |

## 实验 3: 评分标准验证 (L3)

### 3.1 设计

**目的**: 建立 CittaVerse 六维自动评分与人类专家评分的收敛效度

**样本**: 从 Pilot RCT 中随机抽取 100 段叙事

**评估者**: 2 名训练过的评估者 (心理学/语言学背景)

**流程**:
```
100 段叙事 → 2 名评估者独立评分 (6 维度 × 0-100)
                    ↓
              计算评估者间信度 (ICC)
                    ↓
              比较: 自动评分 vs 人工平均分 (Pearson r)
                    ↓
              Bland-Altman 分析 (一致性限)
```

### 3.2 训练评估者

**评估者培训协议**:
1. 阅读六维框架定义 + 评分指南 (2 小时)
2. 练习评分 10 段叙事 + 讨论差异 (2 小时)
3. 独立评分 20 段校准叙事 (1 小时)
4. 达标标准: 校准叙事 ICC > 0.80
5. 如不达标: 追加讨论 + 重新校准

### 3.3 统计分析

| 分析 | 方法 | 目标 |
|------|------|------|
| 评估者间信度 | ICC(2,1) (双向随机，单次测量) | > 0.75 |
| 自动 vs 人工相关 | Pearson r (每个维度 + 综合分) | > 0.70 |
| 一致性分析 | Bland-Altman plot + 95% 一致性限 | 限 ≤ ±15 分 |
| 区分效度 | MCI vs 健康对照组的 d 值 (自动评分) | d > 0.50 |
| 变化敏感性 | 干预前后效应量 (自动评分) | d > 0.30 |

## 实验 4: A/B 测试 — 元记忆增强 (探索性)

### 4.1 设计

**嵌套在 Pilot RCT 干预组内**:
- **A 组** (n=15-20): 标准 AI 引导
- **B 组** (n=15-20): 标准 AI 引导 + 元记忆策略提示

**主要比较**: 六维评分在会话间的改善斜率

### 4.2 元记忆策略

| 策略 | 触发条件 | 提示示例 |
|------|----------|----------|
| **时间锚定** | 时间连贯性 < 60 | "那大概是什么年代的事？" |
| **感官细节** | 事件丰富度 < 60 | "那天的天气/声音/味道还记得吗？" |
| **因果追问** | 因果连贯性 < 60 | "后来为什么会那样？" |
| **情感探索** | 情感深度 < 60 | "回想起来，您当时心里什么感觉？" |
| **自我连接** | 自我认同 < 60 | "这件事对您后来的人生有什么影响？" |

### 4.3 分析方法

- **多层线性模型**: 会话 (Level 1) 嵌套于个体 (Level 2)
- **主要系数**: 组别 × 会话次数 交互项 (反映改善速度差异)
- **效应量**: Cohen's d (组间差异)
