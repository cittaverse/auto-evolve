## 执行摘要

### 核心发现

1. **无成熟"自传体记忆评分"开源项目**：
   - GitHub 检索 `autobiographical memory scoring` → 0 成熟项目
   - 相关技术分散在语音痴呆检测、MRI 分析、对话系统
   - **CittaVerse 可填补空白，建立事实标准**

2. **相关开源项目分类**：
   - **语音痴呆检测**（~5 个项目，50-500⭐）
   - **MRI/神经影像分析**（~10 个项目，100-1000⭐）
   - **对话系统/聊天机器人**（~20 个项目，50-2000⭐）
   - **叙事分析工具**（~5 个项目，20-200⭐）

3. **开源策略趋势**：
   - 学术项目倾向开源（CHI/ACL 等顶会要求）
   - 商业项目倾向闭源（BrainCheck, StoryFile 等）
   - **混合策略兴起**（开源框架 + 闭源模型/数据）

4. **CittaVerse 开源定位机会**：
   - 首个开源自传体记忆评分工具
   - 中文特异性填补国际空白
   - 可吸引学术合作者 + 早期采用者

### Bottom Line

**开源窗口期仍在**：
- 无直接竞品开源项目
- CittaVerse 开源可加速学术采纳
- 建议完善文档 + PyPI 发布，降低使用门槛
