## 执行摘要

### 核心发现

1. **专利布局整体薄弱**：
   - AI 回忆疗法直接相关专利 < 10 篇（基于有限检索）
   - 叙事评估专利几乎空白
   - 主要申请人为大学/研究机构，非商业公司

2. **技术焦点集中**：
   - 对话系统（问题生成、对话管理）
   - 图像检索/生成（记忆触发器）
   - 记忆辅助工具（提醒、时间线）

3. **地域分布**：
   - 美国：BrainCheck 等认知评估专利
   - 欧洲：Universitat Politecnica de Catalunya (Auto-RT 2019)
   - 亚洲：新加坡（Rememo）、中国（待确认）

4. **时间趋势**：
   - 2019-2021：早期探索（对话系统为主）
   - 2022-2024：生成式 AI 兴起（图像生成专利增加）
   - 2025-2026：评估类专利萌芽（BrainCheck 等）

### Bottom Line

**专利窗口期仍在**：
- 核心叙事评估技术无成熟专利壁垒
- CittaVerse 的 6 维度评分框架可申请原创专利
- 建议加速布局（中国 + PCT）

## Part V: 专利与论文协同策略

### 5.1 时间协调

```
┌─────────────────────────────────────────────────────────────┐
│              专利 - 论文协同时间线                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  T0: 技术完成                                               │
│       ↓                                                     │
│  T0+1 周：专利申请提交（确保优先权）                         │
│       ↓                                                     │
│  T0+2-4 周：arXiv 论文提交（公开细节，建立学术存在）          │
│       ↓                                                     │
│  T0+3-6 月：期刊/会议投稿（CHI/JMIR 等）                     │
│       ↓                                                     │
│  T0+12-24 月：专利实质审查                                   │
│       ↓                                                     │
│  T0+24-36 月：专利授权 + 论文发表                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**关键原则**:
1. **专利先于论文公开**: 专利申请提交后再公开技术细节
2. **arXiv 时机**: 专利申请后 2-4 周内提交 arXiv
3. **期刊投稿**: 专利申请后可自由投稿

### 5.2 内容分工

| 内容类型 | 专利文档 | 学术论文 |
|---------|---------|---------|
| **技术细节** | 完整实现（权利要求） | 核心方法 + 实验验证 |
| **理论支撑** | 简略提及 | 深入讨论（文献综述） |
| **实验数据** | 代表性实施例 | 完整实验 + 统计分析 |
| **代码** | 不公开 | 可开源（GitHub） |
| **商业信息** | 不披露 | 不披露 |
