**生成日期**: 2026-03-26  
**分析范围**: AI 回忆疗法 × 叙事评估 × 老年认知健康  
**验证等级**: V1-V2（arXiv API + 公开文献交叉确认）

## 执行摘要

### 核心发现

1. **AI-RT 技术路线分化为三大阵营**：
   - **生成式 AI 派**（Rememo, StoryFile）：聚焦记忆触发器生成
   - **评估式 AI 派**（CittaVerse, BrainCheck）：聚焦认知/叙事质量评估
   - **对话式 AI 派**（Automatic RT 2019, NewDays AI）：聚焦自动化治疗对话

2. **叙事评分技术仍处早期**：
   - arXiv 上"narrative + scoring + LLM"相关论文 125 篇，但针对自传体记忆的<5 篇
   - 现有工作集中于：叙事相似度预测、经济叙事提取、社会感知建模
   - **CittaVerse 的 6 维度自传体记忆评分框架无直接竞品**

3. **专利布局空白**：
   - AI 回忆疗法专利检索结果极少（<10 篇相关）
   - 叙事评估专利几乎空白
   - **窗口期仍在，建议加速专利申请**

4. **开源生态尚未形成**：
   - GitHub 无成熟"autobiographical memory scoring"开源项目
   - 相关技术分散在：语音痴呆检测、MRI 分析、对话系统
   - **CittaVerse 开源策略可建立事实标准**

## Part II: 核心技术模块拆解

### 2.1 记忆触发器生成（生成式 AI 派）

#### Rememo 技术栈（arXiv:2602.17083）

| 模块 | 技术选型 | 实现细节 |
|------|---------|---------|
| **图像生成** | SDXL, Flux.1, Imagen | LoRA 微调（317 张新加坡档案图像） |
| **提示词工程** | LLM-generated prompts | 基于 OCR 识别的物理提示卡 + 自由文本 |
| **多语言支持** | 4 语言（英/华语/马来/泰米尔） | 方言发音标注 |
| **输出形式** | Web 应用 + 打印照片 | 物理 - 数字混合交互 |
| **延迟优化** | 未明确（论文提及延迟问题） | 现场部署才发现延迟 |

**技术优势**：
- LoRA 微调实现文化本地化
- 物理媒介增强情感连接
- AI-in-the-loop 模式（治疗师主导）

**技术局限**：
- 图像生成延迟影响用户体验
- 生成逻辑黑盒，可解释性弱
- 输出非确定性（相同输入→不同图像）

#### StoryFile 技术栈（公开信息）

| 模块 | 技术选型 | 实现细节 |
|------|---------|---------|
| **视频录制** | 专业拍摄设备 | 高质量视频采集 |
| **交互式对话** | 分支叙事 + 视频片段检索 | 预录制视频 + 关键词触发 |
| **内容管理** | 专有 CMS | 家族故事库管理 |
| **定价** | $99-299/月 | B2C 订阅制 |

**技术优势**：
- 高质量视频内容
- 成熟的商业模式
- 品牌认知度高

**技术局限**：
- 非 AI 原生（预录制视频）
- 成本高（专业拍摄）
- 扩展性有限

### 2.2 叙事质量评估（评估式 AI 派）

|------|---------|---------|

| **核心模型** | LLM + 规则系统 | 神经符号 AI 架构 |
| **评分算法** | 加权公式 + 阈值判定 | S/F 等级（S/A/B/C/D/F） |
| **输出形式** | JSON + Gradio UI | 结构化评分 + 可视化报告 |
| **文化适配** | 中文叙事理论 + 本土化阈值 | 针对中国老年人校准 |

**技术优势**：
- 理论驱动的评估框架
- 高可解释性（规则可追溯）
- 输出确定性（相同输入→相同评分）
- 中文特异性优化

**技术局限**：
- 依赖转录质量（ASR 误差传递）
- 实证验证进行中（pilot RCT）
- 学术发表待完成

### 3.2 高相关度论文分析

#### 📄 Cerebra: Multidisciplinary AI Board for Dementia Assessment (arXiv:2603.21597)

**核心贡献**：
- 多智能体 AI 团队协调 EHR、临床笔记、医学影像分析
- 痴呆风险预测 AUROC 0.80（优于单模态 0.74）
- 医生读者研究：准确性提升 17.5 个百分点

**技术方法**：
- 多模态融合（结构化数据 + 文本 + 影像）
- 可解释性仪表盘
- 隐私保护部署（不完整模态鲁棒性）

**对 CittaVerse 启示**：
- 多模态融合是趋势（CittaVerse 当前以文本为主）
- 可解释性对临床接受度至关重要
- 可考虑与神经影像指标关联（长期研究方向）

**验证等级**: V1（arXiv 摘要确认）

---

#### 📄 LoV3D: Longitudinal 3D Brain MRI for Cognitive Prognosis (arXiv:2603.12071)

**核心贡献**：
- 3D 视觉语言模型读取纵向脑 MRI
- 三分类诊断准确率 93.7%（CN/MCI/Dementia）
- 临床加权 Verifier 减少幻觉

**技术方法**：
- 区域级解剖评估 + 纵向比较
- Direct Preference Optimization（无需人工标注）
- 零样本迁移验证（MIRIAD, AIBL 数据集）

**对 CittaVerse 启示**：
- 纵向数据对认知评估价值高
- CittaVerse 的叙事评分可追踪长期变化
- 可探索与 MRI 生物标志物关联

**验证等级**: V1（arXiv 摘要确认）

---

#### 📄 SpeechCARE: Speech-based Cognitive Assessment (arXiv:2511.08132)

**核心贡献**：
- 多模态语音处理（声学 + 语言学特征）
- MCI 检测 AUC 0.90，三分类 AUC 0.88
- SHAP 可解释性 + LLM 推理

**技术方法**：
- Whisper  embeddings + 语言学特征
- Mixture of Experts 动态融合
- LLM 异常检测 + 任务识别

**对 CittaVerse 启示**：
- 语音是重要的认知评估模态
- CittaVerse 可整合声学特征（韵律、停顿等）
- 可解释性模块设计可借鉴

**验证等级**: V1（arXiv 摘要确认）

---

#### 📄 TAI-Speech: Temporal-Aware Iterative Speech Model (arXiv:2510.00030)

**核心贡献**：
- 时序感知迭代框架检测痴呆
- AUC 0.839，准确率 80.6%
- 不依赖 ASR（直接处理音频）

**技术方法**：
- 光流启发的迭代细化（卷积 GRU）
- 跨注意力韵律对齐
- 直接处理语谱图动态

**对 CittaVerse 启示**：
- 时序动态是重要认知标记
- CittaVerse 当前以转录文本为主，可引入声学特征
- 不依赖 ASR 的方法值得探索（减少误差传递）

**验证等级**: V1（arXiv 摘要确认）

---

#### 📄 Demenba: State Space Models for Dementia Detection (arXiv:2507.10311)

**核心贡献**：
- 状态空间模型处理神经心理测试语音
- 细粒度痴呆分类提升 21%
- 训练数据：1000+ 小时（Framingham Heart Study）

**技术方法**：
- 状态空间模型（线性内存/计算扩展）
- 与 LLM 融合增强性能
- 开源代码（匿名仓库）

**对 CittaVerse 启示**：
- 状态空间模型是新兴方向
- 大规模语音数据价值高
- 开源策略可加速学术影响

**验证等级**: V1（arXiv 摘要确认）

### 3.3 技术趋势总结

| 趋势 | 证据 | 对 CittaVerse 启示 |
|------|------|-------------------|
| **多模态融合** | Cerebra, SpeechCARE 等多工作采用 | 整合声学特征 + 文本 + 可能的影像 |

| **大规模验证** | Cerebra 300 万患者、Demenba 1000h 语音 | Pilot RCT 后需扩展验证规模 |
| **开源策略** | Demenba、CAtCh 等开源代码 | 开源可加速学术采纳和事实标准建立 |
