## 2. 产品功能

### 2.1 核心功能

| 功能 | 描述 | 技术实现 |
|------|------|----------|
| **照片扫描** | 上传老照片 | 图像上传 + 存储 |
| **故事生成** | AI 基于照片生成故事 | 多模态 VLM + LLM |
| **时间线整理** | 自动按时间排序 | 图像元数据 + NLP |
| **自传成书** | 生成完整自传 | 长文本生成 + 排版 |
| **打印服务** | 实体书打印 | 合作印刷厂 |

### 2.2 用户体验流程

```
1. 下载 App → 2. 上传照片 → 3. AI 识别内容
                                        ↓
5. 编辑修改 ← 4. AI 生成故事 ← 3. 补充背景信息
        ↓
6. 生成自传 → 7. 可选打印
```

### 2.3 定价模式

| 套餐 | 价格 | 功能 |
|------|------|------|
| **Digital** | £49 一次性 | 数字版自传 (PDF/ePub) |
| **Print** | £99 起 | 数字版 + 实体书 (页数不同) |
| **Premium** | £149 起 | 数字版 + 精装实体书 + 额外副本 |

## 4. 与 CittaVerse 对比

### 4.1 定位对比

| 维度 | MemoryLane | CittaVerse (阿宝) |
|------|------------|------------------|
| **核心目标** | 自传生成 + 家族传承 | 回忆疗法 + 认知健康 |
| **产品定位** | 消费级自传工具 | B2B2C 数字疗法 |
| **目标用户** | 普通家庭、老年人 | MCI/轻度痴呆 + 家属 |
| **输入形式** | 照片上传 | 语音对话 + 照片 |
| **输出形式** | 自传 (数字/实体) | 人生故事书 + 六维评分 |
| **核心价值** | 记录与传承 | 认知干预 + 情感连接 |
| **临床背书** | 无 | Pilot RCT 计划中 |
| **价格** | £49-149 一次性 | 待定价 (预计更低) |

### 4.2 技术对比

| 技术维度 | MemoryLane | CittaVerse | 优劣分析 |
|----------|------------|------------|----------|
| **多模态输入** | ✅ 照片 | ✅ 语音 + 照片 | CittaVerse 更丰富 |
| **AI 深度** | ⚠️ 图像 + 文本生成 | ✅ 叙事评分 + 认知分析 | CittaVerse 更深 |
| **临床验证** | ❌ 无 | 🟡 准备中 | CittaVerse 有潜力 |
| **方言支持** | ❌ 英语为主 | ✅ 中文方言优化 | CittaVerse 更本地化 |
| **输出产物** | 自传书 | 故事书 + 评分报告 | CittaVerse 更功能化 |
| **交互模式** | 单向上传 | 双向对话引导 | CittaVerse 更互动 |

### 4.3 差异化机会

| 机会 | 具体策略 | 优先级 |
|------|----------|--------|
| **临床定位** | 强调认知健康 vs. 纯记录 | P0 |
| **价格优势** | 订阅制 vs. 一次性高价 | P0 |
| **中文优化** | 方言 + 集体记忆 | P0 |
| **叙事评分** | 客观认知评估工具 | P1 |
| **B 端渠道** | 养老机构合作 | P1 |
