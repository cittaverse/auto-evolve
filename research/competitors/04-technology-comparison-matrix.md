## Part III: CittaVerse 定位分析

### 3.1 任务定位对比

```
┌─────────────────────────────────────────────────────────────┐
│                    认知健康 AI 任务图谱                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  诊断分类 ←────────────────────→ 质量评估                   │
│  (What)                        (How Good)                   │
│                                                             │
│  ┌─────────────┐                 ┌─────────────┐           │
│  │ SpeechCARE  │                 │ CittaVerse  │           │
│  │ TAI-Speech  │                 │ (唯一专注者)│           │
│  │ Demenba     │                 │             │           │
│  │ CAtCh       │                 │             │           │
│  │ LoV3D       │                 │             │           │
│  │ xEEGNet     │                 │             │           │
│  └─────────────┘                 └─────────────┘           │
│                                                             │
│  输入：多模态（音频/MRI/EEG）输入：文本（自传体叙事）        │
│  输出：分类标签（HC/MCI/AD）输出：6 维度评分 + S/F 等级       │
│  用户：临床医生            用户：研究人员/产品团队          │
│                                                             │
│  互补关系：诊断分类 × 质量评估 = 完整认知评估               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**关键洞察**:
- CittaVerse 与前沿研究是**互补关系**（非竞争）
- 诊断分类回答"是否有认知障碍"
- 叙事评估回答"叙事质量如何"
- 两者结合可提供更全面的认知健康画像

---

### 3.2 技术路线对比

| 维度 | 诊断分类派 | CittaVerse（评估派） |
|------|-----------|---------------------|
| **核心问题** | Is there impairment? | How good is the narrative? |
| **技术范式** | 深度学习（端到端） | 神经符号（可解释） |
| **输出确定性** | 概率（softmax） | 确定性（规则判定） |
| **可解释性** | 后验（SHAP/LIME） | 先验（规则可追溯） |
| **数据需求** | 大规模标注数据 | 中等规模 + 理论驱动 |
| **文化适配** | 重新训练/微调 | 阈值校准 |
| **临床接受** | 黑盒疑虑 | 透明决策 |

**CittaVerse 优势**:
- 可解释性更强（规则可追溯）
- 数据需求更低（理论驱动）
- 文化适配更灵活（阈值校准）
- 临床接受度可能更高（透明决策）

**CittaVerse 劣势**:
- 诊断预测能力弱（非核心任务）
- 多模态融合滞后
- 大规模验证不足
