### 1.4 CittaVerse 专利申请策略

#### 核心专利：6 维度叙事评分方法

**专利名称**: "基于六维度自传体记忆理论的叙事质量自动评估方法"

**独立权利要求**:

一种基于六维度自传体记忆理论的叙事质量自动评估方法，其特征在于，包括以下步骤：

1. **输入接收步骤**: 接收用户的自传体叙事文本，所述叙事文本由语音识别转录或直接文本输入生成；

2. **维度提取步骤**: 使用大语言模型 (LLM) 从所述叙事文本中提取以下六个维度的特征：
   - 内部细节 (Internal Details): 感知、情感、思维等主观体验描述
   - 外部细节 (External Details): 时间、地点、人物等客观事实描述
   - 事件分段 (Event Segmentation): 叙事中的事件边界识别
   - 情感效价 (Emotional Valence): 叙事的情感倾向（正面/负面/中性）
   - 叙事连贯性 (Narrative Coherence): 因果连接词、时间连接词的使用
   - 时间一致性 (Temporal Consistency): 时间线索的逻辑一致性；

3. **规则评分步骤**: 基于预定义的规则系统对每个维度进行评分，所述规则包括：
   - 内部细节计数规则：统计内部细节关键词的出现次数
   - 外部细节计数规则：统计外部细节关键词的出现次数
   - 事件分段评分规则：基于事件边界密度判定
   - 情感效价评分规则：基于情感词典计算效价分数
   - 叙事连贯性评分规则：基于连接词密度和类型计算
   - 时间一致性评分规则：检测时间冲突并扣分；

4. **等级判定步骤**: 根据六个维度的加权总分，判定叙事质量等级为 S/A/B/C/D/F 之一；

5. **报告生成步骤**: 生成可解释的评分报告，包含各维度得分、等级判定依据、改进建议。

**从属权利要求**（部分）:

- **中文特异性**: 中文情感词典、连接词库、历史事件映射表
- **阈值校准**: 不同教育水平/地域/方言区的阈值调整
- **神经符号架构**: LLM 提取 + 规则评分的混合架构
- **纵向追踪**: 多次叙事评分变化追踪
- **多模态输入**: 音频声学特征融合

**申请地域**:
- 中国（CNIPA）：核心市场
- PCT：保留国际申请权
- 美国（USPTO）：3 年内根据商业化进展决定

**预计成本**:
- 中国发明专利：¥15,000-25,000（含代理费）
- PCT 申请：$5,000-8,000
- 美国专利：$15,000-25,000（后续）
- **总计（3 年）**: ~¥430,000

**时间线**:

- 2026-05: 中国专利申请提交

```
│  │  无自动评估功能                                      │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 核心技术

| 模块 | 技术实现 | 创新点 |
|------|---------|-------|
| **图像生成** | LoRA 微调 Stable Diffusion | 文化/年代本地化 |
| **治疗师工作流** | AI-in-the-loop（非 AI-only） | 增强人类关系性工作 |
| **物理 - 数字混合** | 打印照片 + 数字生成 | 适合老年用户习惯 |

#### 研究结果

- **参与者**: N=21 痴呆患者 + 5 照护者
- **研究周期**: 2 周实地研究
- **核心发现**: AI 应增强而非替代治疗师的关系性工作
- **局限**: 仅关注 session 前准备，无 session 后评估

| 维度 | Rememo | CittaVerse |
|------|--------|------------|
| **定位** | Session 前准备 | Session 后评估 |
| **核心技术** | 图像生成 (LoRA SD) | 叙事评分 (LLM+ 规则) |

| **商业化** | 研究阶段 | Pilot RCT 进行中 |

**结论**: **互补关系，非竞争**。合作机会：
- 完整工作流：Rememo (前) + CittaVerse (后) = 端到端 RT 工具链
