## 2. 产品功能

### 2.1 核心功能

| 功能 | 描述 | 技术实现 |
|------|------|----------|
| **视频问答** | 用户录制视频回答预设问题 | 视频录制 + 存储 |
| **AI 生成传记** | 基于视频内容生成文字传记 | ASR + NLP 摘要 |
| **交互式对话** | 后人可与"数字逝者"对话 | 视频片段检索 + 播放 |
| **家族共享** | 家庭成员共享故事库 | 权限管理 + 云存储 |
| **纪念页面** | 创建在线纪念页面 | Web 展示 |

### 2.2 用户体验流程

```
1. 注册账号 → 2. 选择问题模板 → 3. 录制视频回答
                                          ↓
5. 生成传记 ← 4. AI 处理 (ASR + 摘要) ← 3. 重复录制
        ↓
6. 家族共享/纪念页面
```

### 2.3 定价模式

| 套餐 | 价格 | 功能 |
|------|------|------|
| **Basic** | $99/月 | 基础视频存储、有限问题模板 |
| **Premium** | $199/月 | 无限视频、AI 传记、家族共享 |
| **Legacy** | $299/月 | 全部功能 + 纪念页面 + 优先支持 |

## 4. 与 CittaVerse 对比

### 4.1 定位对比

| 维度 | StoryFile | CittaVerse (阿宝) |
|------|-----------|------------------|
| **核心目标** | 视频传记 + 数字遗产 | 回忆疗法 + 认知健康 |
| **产品定位** | 消费级视频传记 | B2B2C 数字疗法 |
| **目标用户** | 高净值家庭 | MCI/轻度痴呆 + 家属 |
| **输入形式** | 视频录制 | 语音对话 + 照片 |
| **输出形式** | 视频库 + 文字传记 | 人生故事书 + 六维评分 |
| **核心价值** | 记录与传承 | 认知干预 + 情感连接 |
| **临床背书** | 无 | Pilot RCT 计划中 |
| **价格** | $99-299/月 | 待定价 (预计更低) |

### 4.2 技术对比

| 技术维度 | StoryFile | CittaVerse | 优劣分析 |
|----------|-----------|------------|----------|
| **多模态输入** | ⚠️ 视频 | ✅ 语音 + 照片 | CittaVerse 更轻量 |
| **AI 深度** | ⚠️ ASR + 摘要 | ✅ 叙事评分 + 认知分析 | CittaVerse 更深 |
| **临床验证** | ❌ 无 | 🟡 准备中 | CittaVerse 有潜力 |
| **方言支持** | ⚠️ 英语为主 | ✅ 中文方言优化 | CittaVerse 更本地化 |
| **输出产物** | 视频库 + 传记 | 故事书 + 评分报告 | 不同定位 |
| **交互模式** | 单向录制 | 双向对话引导 | CittaVerse 更互动 |

### 4.3 差异化机会

| 机会 | 具体策略 | 优先级 |
|------|----------|--------|
| **临床定位** | 强调认知健康 vs. 纯记录 | P0 |
| **价格优势** | 更低价格 + 医保可能性 | P0 |
| **中文优化** | 方言 + 集体记忆 | P0 |
| **叙事评分** | 客观认知评估工具 | P1 |
| **B 端渠道** | 养老机构合作 vs. 纯 C 端 | P1 |
