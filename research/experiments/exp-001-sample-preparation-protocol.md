## 二、正常叙述样本 (140 条)

### 2.1 来源分布

| 数据集 | 目标数量 | 主题分布 | 长度分布 |
|--------|----------|----------|----------|
| **阿宝会话** | 60 | 童年记忆 20 + 工作经历 20 + 家庭生活 20 | 短 20 + 中 30 + 长 10 |
| **高级谈** | 50 | 童年记忆 15 + 工作经历 20 + 家庭生活 15 | 短 15 + 中 25 + 长 10 |
| **SeniorTalk** | 30 | 童年记忆 10 + 工作经历 10 + 家庭生活 10 | 短 10 + 中 15 + 长 5 |

### 2.2 筛选标准

**纳入标准**:
- ✅ 中文叙事文本 (允许方言词)
- ✅ 字数 100-1000 字
- ✅ 主题明确 (童年/工作/家庭/重大事件)
- ✅ 无明显语法错误或机器翻译感

**排除标准**:
- ❌ 字数 < 50 或 > 1500
- ❌ 明显编造/虚构痕迹
- ❌ 纯事实罗列无情感表达
- ❌ 包含敏感 PII (姓名/地址/电话未脱敏)

### 2.3 抽样方法

```python
# 伪代码：分层随机抽样
import random

def sample_normal_narratives():
    # 从各数据集按主题和长度分层
    datasets = {
        'abao': load_dataset('abao_conversations'),
        'gaojitalk': load_dataset('gaojitalk'),
        'seniortalk': load_dataset('seniortalk')
    }
    
    samples = []
    for dataset, target_count in [('abao', 60), ('gaojitalk', 50), ('seniortalk', 30)]:
        # 按主题分层
        by_theme = group_by_theme(datasets[dataset])
        for theme, count in theme_distribution.items():
            theme_samples = random.sample(by_theme[theme], count)
            # 按长度分层
            by_length = group_by_length(theme_samples)
            for length, lcount in length_distribution.items():
                samples.extend(random.sample(by_length[length], lcount))
    
    return samples[:140]
```

### 2.4 脱敏处理

**必须脱敏的 PII**:
- 姓名 → `[姓名]` 或 `[亲属称谓]`
- 地址 → `[地点]` 或 `[城市名]`
- 电话/身份证 → `[联系方式]`
- 具体单位名 → `[单位]`

**保留的信息**:
- 年代/年份 (如 "1985 年")
- 季节/时间 (如 "夏天"、"小时候")
- 泛化地点 (如 "杭州"、"公园")

## 三、边界案例样本 (40 条)

### 3.1 筛选标准

**条件 A: L0 置信度低**
- L0 置信度评分 < 0.6
- 原因可能：信息稀疏、表达模糊、维度冲突

**条件 B: L0 评分中等**
- L0 综合评分 55-75 分
- 原因可能：各维度评分差异大、规则命中不充分

### 3.2 来源

| 来源 | 目标数量 | 筛选方式 |
|------|----------|----------|
| **L0 历史评分日志** | 30 条 | 查询置信度 < 0.6 的样本 |
| **人工识别** | 10 条 | 研究团队主观判断为"难以评定"的样本 |

### 3.3 预期特征

| 特征类型 | 示例 | 预期数量 |
|----------|------|----------|
| **信息稀疏** | "那天去了公园，玩了很久，很开心" | 10 条 |
| **表达模糊** | "好像是 80 年代，记不清了，反正很高兴" | 10 条 |
| **维度冲突** | 内部细节丰富但外部细节缺失 | 10 条 |
| **长度极端** | <100 字或 >1000 字 | 10 条 |

## 四、堆砌样本 (20 条)

### 4.1 构造原则

模拟 Reward Hacking 攻击：刻意堆砌评分维度关键词，但叙事质量低。

### 4.2 堆砌类型分布

| 堆砌类型 | 构造方式 | 样本量 | 预期检测信号 |
|---------|---------|--------|-------------|
| **C1 堆砌 (感官)** | 每 50 字插入 1 个感官细节词 | 5 条 | 信息密度异常高 |
| **C2 堆砌 (外部)** | 每 50 字插入 1 个外部实体 | 5 条 | 外部细节密度 >3 倍期望 |
| **C4 堆砌 (情感)** | 密集使用情感词，与事件效价不匹配 | 5 条 | 情感 - 事件不一致 |
| **混合堆砌** | 同时堆砌多个维度关键词 | 5 条 | 叙事多样性下降 |

### 4.3 构造示例

#### 示例 1: C1 堆砌 (感官细节)
```
那天我看到阳光照在窗台上，我听到鸟叫声，我闻到花香，
我看到妈妈在厨房忙碌，我听到锅铲的声音，我闻到饭菜香，
我看到弟弟在玩耍，我听到他的笑声，我感到很温暖，
我看到墙上的钟，我听到滴答声，我闻到肥皂的味道...
```

**特征**: 感官词密度异常高 (每句都有"看到/听到/闻到")，但缺乏事件连贯性。

#### 示例 2: C2 堆砌 (外部细节)
```
1985 年夏天，在杭州西湖边，我和母亲、父亲、弟弟一起去了公园。
那天是 7 月 15 日，星期六，下午 3 点。公园里有很多人，
有老人、小孩、年轻人。我们看到了荷花、柳树、假山、亭子...
```

**特征**: 时间/地点/人物/物体密集堆砌，但缺乏情感表达和叙事深度。

#### 示例 3: C4 堆砌 (情感词)
```
那天我非常开心，特别激动，感到无比幸福，心情很好，
虽然遇到了一些困难让我有点难过，但总体来说很高兴，
朋友们都很快乐，我也很兴奋，真是愉快的一天...
```

**特征**: 情感词密集但空洞，与具体事件不匹配。

#### 示例 4: 混合堆砌
```
1985 年 7 月 15 日，在杭州西湖边，我看到荷花，闻到花香，
听到鸟叫，感到非常开心。母亲、父亲、弟弟都在，
我们看到了柳树、假山、亭子，听到笑声，闻到饭菜香，
感到幸福、快乐、激动、温暖...
```

**特征**: 同时堆砌感官、外部、情感关键词，叙事支离破碎。

### 4.4 构造流程

1. **确定堆砌类型** (C1/C2/C4/混合)
2. **选择基础叙事框架** (简单事件)
3. **按规则插入堆砌词** (每 50 字插入 1 个)
4. **人工审核自然度** (确保可读但可检测)
5. **编号归档** (STUFFED-001 至 STUFFED-020)

## 五、样本文件格式

### 5.1 CSV 结构

```csv
sample_id,text,layer,theme,length_chars,source,notes
NORMAL-001,"小时候...",normal,childhood,350,abao,脱敏完成
NORMAL-002,"工作时...",normal,work,520,gaojitalk,脱敏完成
...
BORDER-001,"那天...",border,ambiguous,180,log,L0 置信度 0.45
...
STUFFED-001,"那天我看到...",stuffed,C1,400,constructed,C1 感官堆砌
...
```

### 5.2 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `sample_id` | string | 唯一标识符 (NORMAL/BORDER/STUFFED-XXX) |
| `text` | string | 叙事文本 (脱敏后) |
| `layer` | string | 样本层级 (normal/border/stuffed) |
| `theme` | string | 主题 (childhood/work/family/major_event/ambiguous) |
| `length_chars` | int | 字符数 (不含标点) |
| `source` | string | 来源数据集或 "constructed" |
| `notes` | string | 备注 (脱敏状态/堆砌类型/L0 置信度等) |

## 六、质量控制

### 6.1 正常叙述样本

- [ ] 主题分布符合目标比例 (童年/工作/家庭/重大事件 ≈ 均衡)

- [ ] 每种堆砌类型 5 条
