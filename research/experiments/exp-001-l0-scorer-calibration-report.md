### 2.4 等级边界

| 等级 | 分数范围 | 当前行为 | 问题 |
|------|---------|---------|------|
| S | 90-100 | 正常 | ✅ |
| A | 80-89 | 正常 | ✅ |
| B | 70-79 | 正常 | ✅ |
| C | 60-69 | 正常 | ✅ |
| D | 0-59 | 正常 | ✅ |

**边界防御问题** (来自 2026-03-25 鲁棒性测试 #1):
- `assign_grade(-1)` 返回 D (不崩溃但语义不严谨)
- `assign_grade(101)` 返回 S (同上)

**建议修复**:
```python
def assign_grade(score: float) -> str:
    score = max(0, min(100, score))  # clamp to [0, 100]
    if score >= 90: return 'S'
    elif score >= 80: return 'A'
    elif score >= 70: return 'B'
    elif score >= 60: return 'C'
    else: return 'D'
```

## 三、边界情况处理

### 3.1 6 种极端场景测试结果

| 场景 | 当前行为 | 评级 | 建议 |

| **空文本** | 返回零分，不崩溃 | ✅ 通过 | 无需修改 |

| **无意义文本** | 返回低分 | ✅ 通过 | 无需修改 |
| **混合语言** | 处理正常 | ✅ 通过 | 无需修改 |

**验证状态**: ✅ V4 (动态验证)

### 3.2 已知脆弱点 (来自鲁棒性测试 #1)

| ID | 脆弱点 | 严重性 | 根因 | 修复状态 |
|----|--------|--------|------|---------|
| V1 | 情感词堆砌可欺骗唤醒度检测器 | Medium | 纯词频统计，无结构验证 | ⏳ v0.6 待修复 |
| V2 | 最大化构造可达 100 分 | Medium | 各维度独立计算，无跨维度一致性校验 | ⏳ v0.6 待修复 |
| V3 | assign_grade 不防御超范围输入 | Low | 无输入 clamp | ⏳ v0.7 待修复 |

**V1 修复方案 (v0.6)**:
1. 增加"叙事结构校验层"：唤醒度评分前先检查文本是否包含至少 1 个事件结构（主谓宾）
2. 增加"词汇多样性惩罚"：同一情感词重复 n 次时边际贡献递减
3. 长期方案：LLM judge 替代规则引擎，few-shot 示例包含对抗样本

**V2 修复方案 (v0.6)**:
1. 增加"内容真实性分数"维度（文本中是否有具体细节而非泛指）
2. 各维度增加"质量门槛"：词频密度超过阈值时饱和/惩罚
3. 交叉验证：事件描述与文本实际内容的语义一致性

**验证状态**: ✅ V4 (动态验证 — 脆弱点已识别)

## 四、评分一致性分析

### 4.1 跨叙事类型稳定性

基于 50 样本测试 (5 种类型 × 10 样本)：

| 叙事类型 | 样本数 | 均值 | 标准差 | 范围 | 解读 |
|---------|--------|------|--------|------|------|
| 正向 | 10 | 80.16 | 1.48 | 77.93-82.67 | 稳定 |
| 负向 | 10 | 82.21 | 0.81 | 81.37-83.73 | **最稳定** |
| 中性 | 10 | 57.53 | 1.33 | 55.77-59.20 | 稳定，但分数偏低 |
| 反思性 | 10 | 80.74 | 1.46 | 78.60-83.63 | 稳定 |
| 创伤性 | 10 | 83.57 | 1.21 | 81.30-85.93 | 稳定，分数最高 |

**关键发现**:
- 创伤性叙述评分最高 (83.57)，符合心理学预期 (创伤记忆更生动)
- 中性叙述评分最低 (57.53)，符合预期 (缺乏情感/细节)
- 所有类型的标准差 < 1.5，**跨类型稳定性良好**
