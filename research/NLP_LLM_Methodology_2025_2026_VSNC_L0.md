## Key Findings

### 1. EM-LLM：人類啟發式情景記憶架構 (ICLR 2025)

**核心創新**：
- **事件分段**：使用貝葉斯驚奇度 (Bayesian Surprise) + 圖論邊界優化，將連續 token 流在線分段為連貫的「情景事件」
- **兩階段檢索**：相似度檢索 + 時間相鄰性檢索，模擬人類記憶存取模式
- **無需微調**：可直接集成到現有 LLM，無需 fine-tuning
- **千萬級 token 檢索**：在 10M tokens 尺度上成功檢索，這是全上下文模型計算不可行的

**性能表現**：
- LongBench 和 ∞-Bench 基準測試中持續超越 SOTA (InfLLM)
- 在檢索和 QA 任務上提升最高可達 40%
- 事件分段與人類感知事件邊界高度相關

**VSNC/L0 適用性**：⭐⭐⭐⭐⭐ (極高)
- 可直接應用於用戶人生故事的分段與組織
- 支持跨會話的長期記憶檢索
- 事件邊界檢測與人類直覺一致，適合回憶敘事場景

**來源**: Fountas et al., ICLR 2025, arXiv:2407.09450v3

---

### 2. 持久化記憶 + 用戶畫像框架 (arXiv 2025)

**核心創新**：
- **統一個性化定義**：將個性化作為概念基礎，推導自適應 LLM Agent 的技術需求
- **持久化記憶**：整合多源檢索、動態協調、自驗證機制
- **演進式用戶畫像**：支持長期交互中的用戶偏好/狀態演化
- **五日用戶研究**：初步驗證用戶對感知個性化的反饋

**評估指標**：
- 檢索準確率、回應正確性、BertScore
- 用戶研究提供感知個性化的早期洞察

**VSNC/L0 適用性**：⭐⭐⭐⭐⭐ (極高)
- 直接對應阿寶/小念的用戶畫像系統
- 支持跨會話的個性化記憶演化
- 可與 EM-LLM 互補：EM-LLM 管事件分段，此框架管用戶適配

**來源**: Westhäußer et al., arXiv:2510.07925 (2025-10)

---

### 3. Eye2Recall：眼動追蹤 + LLM 回憶交互 (arXiv 2025/2026)

**核心創新**：
- **混合主動交互**：整合眼動追蹤 (視覺興趣檢測) + 自然語言交互
- **照片回憶場景**：專為老年人照片驅動的回憶療法設計
- **專家訪談 + 用戶研究**：10 位老年參與者評估
- **可及性設計**：對齊老年人自然交互模式

**設計考量**：
- 老年人使用 LLM 驅動回憶體驗的挑戰
- 凝視 + 語音作為混合主動交互的潛力
- 增強積極老齡化的賦能技術

**VSNC/L0 適用性**：⭐⭐⭐⭐ (高)
- 直接對應照片回憶功能 (abao-photo-recall skill)
- 為多模態交互 (視覺 + 語音) 提供設計參考
- 中國老年用戶可及性設計可借鑒

**來源**: Han et al., arXiv:2508.02232v2 (2025-08 提交，2026-02 修訂)

### 4. 記憶敘事 NLP 分析方法論 (Trends in Cognitive Sciences 2025)

**核心發現**：
- **NLP 方法多樣性**：認知神經科學研究開始使用 NLP 檢查記憶敘事
- **方法選擇挑戰**：NLP 方法的多樣性使研究者難以選擇合適技術
- **跨群組/任務比較**：目標是理解記憶回憶差異的潛在機制

**VSNC/L0 適用性**：⭐⭐⭐⭐ (高)
- 為敘事質量評估提供方法論框架
- 支持內部/外部細節自動評分 (對應 abao-recall-miner)
- 可借鑒認知科學的驗證方法

**來源**: Trends in Cognitive Sciences, 2025 (多篇相關論文)

### 5. 痴呆症 NLP 應用系統回顧 (TACL 2025)

**核心發現**：

- **關鍵研究領域**：痴呆症檢測、語言生物標記、遠程監測
- **語音分析**：成本效益高於傳統腦成像方法

**VSNC/L0 適用性**：⭐⭐⭐ (中)

- 語音/語言模式分析技術可复用
- 需注意醫療合規邊界

**來源**: TACL, 2025 (系統回顧)

   - 借鑒記憶敘事 NLP 方法論，升級 abao-recall-miner

   - 情景記憶分段與人類直覺一致，適合講故事場景
