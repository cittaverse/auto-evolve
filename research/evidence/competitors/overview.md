### 1. Rememo (⭐⭐⭐ 最直接竞品)

**来源**: arXiv:2602.05051 (2026-02-19)  
**定位**: AI-in-the-loop 治疗师工具，支持痴呆症回忆疗法

**关键特性**:
- Therapist-oriented design (治疗师主导，非直接 ToC)
- LLM 引导回忆对话
- 会话记录与进展追踪
- 临床工作流集成

**对 CittaVerse 的启示**:
- ✅ 验证了 AI+ 回忆疗法的临床价值
- ✅ 治疗师-in-the-loop 模式降低风险
- ⚠️ 需差异化：CittaVerse 更侧重自主叙事 + 量化评估

## 竞争格局总结

### 市场空白点

1. **中文老年口语优化**: 现有工具多为英文，缺乏中文语境适配
2. **叙事质量量化评估**: 多数产品停留在"记录"，缺乏结构化评估
3. **神经符号架构**: 纯神经或纯规则，缺少混合架构开源实现
4. **临床验证闭环**: 消费级产品缺乏临床证据，学术工具缺乏产品化

### CittaVerse 定位

```
┌─────────────────────────────────────────────────────┐
│              老年认知健康 AI 产品矩阵                │
├─────────────────┬─────────────────┬─────────────────┤
│   学术/研究工具   │   CittaVerse    │   消费级产品    │
│   (LLM-MCI,     │   (神经符号架构  │   (StoryFile,   │
│    LLMCARE)     │    + 中文优化    │    MemoryLane)  │
│                 │    + 临床验证)   │                 │
├─────────────────┼─────────────────┼─────────────────┤
│ • 纯英文         │ • 中文优先       │ • 体验优先       │
│ • 检测非评估     │ • 评估 + 引导    │ • 记录非评估     │
│ • 无产品化       │ • 产品化闭环     │ • 无临床验证     │
└─────────────────┴─────────────────┴─────────────────┘
```
