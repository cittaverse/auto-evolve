## Evidence Grading System

| Grade | Description | Example |
|-------|-------------|---------|
| **V4** | Dynamic verification / Reproducible | Ran experiment, got results |
| **V3** | Static verification | Checked source code, config, docs |
| **V2** | Multi-source corroboration | 2+ independent sources |
| **V1** | Single source confirmation | One paper, one page |
| **V0** | Unverified / Inference only | Based on experience, pattern |
