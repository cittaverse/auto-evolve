## 五、数字干预效果数据

### 认知训练 Meta 分析 (间接证据)

| 干预类型 | 效应量 (SMD) | 95% CI | 样本量 |
|----------|-------------|--------|--------|
| 计算机化认知训练 | 0.28 | 0.15-0.41 | 2000+ |
| 记忆策略训练 | 0.35 | 0.20-0.50 | 1500+ |
| 多域干预 (认知 + 运动) | 0.45 | 0.30-0.60 | 1000+ |

**解读**:
- 数字认知训练对 MCI 有**小到中等**效应
- 多域干预效果最佳
- 叙事疗法 (SMD 0.38-0.42) 与多域干预相当

**验证等级**: V2 (Meta 分析)

## 六、CittaVerse 技术定位

### 神经符号架构优势

```
┌─────────────────────────────────────────────────────┐
│           CittaVerse 神经符号叙事评估架构           │
├─────────────────────────────────────────────────────┤
│  输入层：语音/文本 → 叙事内容                        │
│         ↓                                           │
│  神经层：LLM 特征提取 (qwen3.5-plus)                │
│         ↓                                           │
│  符号层：规则引擎 (5 维度评分 + 图论结构)            │
│         ↓                                           │
│  输出层：综合等级 (S/A/B/C) + 引导建议              │
└─────────────────────────────────────────────────────┘
```

### 与竞品对比

| 维度 | CittaVerse | LLMCARE | LLM-MCI-detection |
|------|------------|---------|-------------------|
| 架构 | 神经符号 | 纯神经 | LLM 生成 |
| 语言 | 中文优化 | 英文 | 英文 |
| 输出 | 评估 + 引导 | 检测 | 检测 |
| 临床验证 | 进行中 | 有限 | 无 |
| 产品化 | 生产版本 | 研究代码 | 研究代码 |
