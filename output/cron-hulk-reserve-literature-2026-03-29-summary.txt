### Pillar A: 自传体记忆 LLM 评分

**黄金标准**: Mistica et al. (2025), Behavior Research Methods
- Fine-tuned LLaMA-3 (8B)
- 与人类评分者相关性 **r = 0.87**
- 输出：internal/external details 连续评分
- 开源：Jupyter Notebook 可用

**VSNC L0 对标状态**:
```
v0.5.1-final (03-28): ✅ 131/131 鲁棒性测试通过
v0.6 规划：LLM 仲裁增强（置信度<0.7 触发）
v0.7 目标：人工标注 50 条，r > 0.80
v0.8 目标：人工标注 100 条，r > 0.87（黄金标准）
```

**关键方法**:
- 7 维度评分：事件连贯性、情感深度、感官细节、时间定位、自我认同、信息密度、情绪唤醒度
- 动态理想比例：ideal_central_ratio = 0.50 + (arousal - 3) * 0.05
- 神经符号架构：规则层（可解释）+ LLM 仲裁（灵活性）

**验证等级**: V2 (Springer + PubMed + ResearchGate 交叉确认)

### Pillar B: 语音生物标志物

**临床背书**:
1. **Nature Communications Medicine (2025)**
   - 语音作为认知障碍筛查生物标志物
   - 早于临床症状数年检测
   - 可扩展、非侵入、低成本

2. **NIA + Alzheimer's & Dementia (2025)**
   - AI 语音分析预测 AD 转化准确率 **>78%** (6 年内)
   - 大规模临床验证

3. **PROCESS Challenge 2025 (Interspeech)**
   - 国际竞赛推动标准化
   - 3 项临床任务 + 多模态特征（声学 + 文本 + LLM macro-descriptors）
   - 最佳系统 = 互补模型组合

**6 种具体 biomarkers** (Robot Speech Biomarkers, arXiv 2025):
1. Altered Grammar
2. Pragmatic Impairments
3. Anomia
4. Disrupted Turn-Taking
5. Slurred Pronunciation
6. Prosody Changes

**Composite biomarker score** 优于单个标志物，与 MMSE 中度相关。

**VSNC 集成路径**:
```
Phase 1 (P0): ASR 选型验证
├─ 测试讯飞/阿里 ASR 对老年语音的 WER/CER
├─ 验证是否针对老年/构音障碍优化
└─ 收集 Pilot RCT 语音作为领域适配数据

Phase 2 (P1): 声学特征提取
├─ 集成 openSMILE/eGeMAPS 特征
├─ 探索语音 biomarkers 作为增值功能
└─ 需临床验证 + 合规审批

Phase 3 (P2): 多模态融合
├─ 语音 biomarkers + LLM 叙事评分
├─ 参考 PROCESS Challenge 多模型融合策略
└─ 参加 PROCESS Challenge 2026 验证
```

**验证等级**: V2 (Nature + NIA + Interspeech 交叉确认)

### Pillar D: 竞品与架构参考

#### Rememo (CHI 2026, arXiv:2602.05051)

**最直接竞品对标**:
- 定位：AI-in-the-loop 治疗师工具（非替代，是增强）
- 场景：痴呆症回忆疗法
- 输入：照片 → 生成问题
- 核心理念：AI 支持人类引导者，尊重关系动态
- 语境：新加坡多元文化

**VSNC 差异化**:
| 维度 | Rememo | CittaVerse |
|------|--------|------------|
| 定位 | 会话前准备（生成记忆触发器） | 会话后评估（叙事质量评分）+ 人生故事书 |
| 技术 | 生成式 AI (SDXL/Flux.1) | 神经符号 AI (规则+LLM) |
| 用户 | 治疗师/护理人员 | 研究人员/临床医生 + C 端家庭 |
| 文化 | 新加坡 | 中国（方言、集体记忆锚点） |
| 输出 | 会话引导建议 | 叙事质量评分 + 人生故事书 |

**CHI 2026 倒计时**: 16 天 (04-13) — 需监测正式发表

**验证等级**: V2 (arXiv + 摘要交叉确认)

#### Sophia System 3 (arXiv:2512.09560)

**叙事记忆架构参考**:
- System 3 = 元认知层，凌驾于 System 1/2 之上
- 负责叙事身份 (narrative identity) 和长周期适应
- 四大协同机制：process-supervised thought search、**narrative memory**、user/self modeling、hybrid reward system
- 高复杂度任务成功率提升 40%

**VSNC 启示**:
- 叙事记忆作为 agent 架构核心组件（非简单 RAG）
- 身份连续性通过叙事记忆维持
- "人生故事书"不仅是产品输出，也是 agent 架构组件

**验证等级**: V2 (arXiv 确认)

#### ACP (Agentic Cognitive Profiling, arXiv:2603.17392)

**多 agent 认知评估框架**:
- 4 评委 + 仲裁 = 5 agent 协作
- 与人类专家评分匹配率 **90.5%**
- 与 VSNC 多 agent scoring pipeline 最直接对标

**验证等级**: V2 (arXiv 确认)
