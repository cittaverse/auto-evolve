## 统一框架：四阶段模块化设计

论文提出 LLM Agent 记忆系统的统一框架，分解为四个核心组件：

### ❶ 信息提取 (Information Extraction)

| 方法 | 描述 | 代表系统 |
|------|------|---------|
| Direct Archiving | 直接归档原始消息 + 时间戳 | MemoryBank, MemGPT, MemoChat |
| Summarization-based | LLM 生成摘要/关键词/标签 | A-MEM, Mem0, MemOS |
| Graph-based | 抽取实体 - 关系三元组构建知识图谱 | Mem0g, Zep |

**VSNC 映射**: REMem 事件分段 → Direct Archive + Graph-based 混合 (事件边界 + 语义三元组)

---

### ❷ 记忆管理 (Memory Management) — 5 种核心操作

| 操作 | 人类记忆类比 | 实现范式 | 代表系统 |
|------|-------------|---------|---------|
| **Connecting** | 关联相关经历 | Associative Links / Structural Edges | A-MEM, Zep, MemTree |
| **Integrating** | 整合碎片记忆 | Summarization / Abstraction | MemoryBank, MemoChat, MemoryOS |
| **Transforming** | 短时→长时迁移 | Stage-wise Transfer / Community Formation | MemoryOS, MemGPT, Zep |
| **Updating** | 更新旧记忆 | Rule-based / LLM-based / Agent-based | MemoryBank, MemTree, MemGPT |
| **Filtering** | 过滤过时信息 | Usage-based / Content-based | MemoryOS, Mem0, Mem0g |

**VSNC 映射**:
- CittaVerse 四层架构 (Working/Episodic/Semantic/Procedural) → **Transforming** 操作的典型实例
- REMem 图构建 → **Connecting** (结构边) + **Integrating** (社区形成)
- 多 Agent 评分 Pipeline → **Updating** (Agent-based) + **Filtering** (Content-based)

---

### ❸ 记忆存储 (Memory Storage)

| 维度 | 类型 | 描述 | 代表系统 |
|------|------|------|---------|
| **组织中心** | Flat | 统一单层存储 (FIFO/JSON) | MemoChat, Mem0 |
| | Hierarchical | 多层分区 (短/中/长期) | MemoryOS (3 层), MemGPT |
| **表征中心** | Vector-based | 嵌入 + 向量索引 (FAISS/Qdrant) | A-MEM, MemoryBank, Mem0 |
| | Graph-based | 树/知识图谱/时序图 | MemTree, Zep, Mem0g |

**VSNC 映射**:
- CittaVerse Working Memory → Hierarchical (Short-term)
- Episodic Memory → Graph-based (事件图)
- Semantic Memory → Vector-based (语义嵌入)
- Procedural Memory → Hierarchical (Long-term, 规则/技能)

---

### ❹ 信息检索 (Information Retrieval)

| 范式 | 机制 | 优势 | 代表系统 |
|------|------|------|---------|
| Lexical-Based | 词面重叠 (Jaccard/BM25) | 精确匹配名称/实体 | — |
| Vector-Based | 语义相似度 (余弦/HNSW) | 解决词汇不匹配 | A-MEM, MemoryBank, Mem0 |
| Structure-Based | 图遍历/多跳推理 | 捕获关系结构 | Mem0g, Zep |
| LLM-Assisted | LLM 主动推理/查询改写 | 潜在语义依赖 | MemoChat |

**VSNC 映射**: 
- 叙事评分检索 → Vector-Based (语义相似度) + Structure-Based (事件图多跳)
- 回忆触发器 → LLM-Assisted (查询改写) + Vector-Based

## 实验结果：10 种方法在两个 Benchmark 上的对比

### LONGMEMEVAL (多会话长程记忆)

| 方法 | Overall F1 | Overall BLEU-1 | 关键优势维度 |
|------|-----------|---------------|-------------|
| **MemoryOS** | **46.04** | **39.42** | Multi-Session (75.10 F1) |
| MemTree | 44.25 | 37.02 | Information Extraction (68.09 F1) |
| MemOS | 39.88 | 32.02 | Information Extraction (69.06 F1) |
| Mem0 | 37.85 | 32.62 | Information Extraction (67.48 F1) |
| Mem0g | 38.47 | 33.25 | Knowledge Updates (47.15 F1) |
| MemoryBank | 35.79 | 30.70 | Multi-Session (57.41 F1) |
| MemGPT | 32.63 | 25.82 | — |
| A-MEM | 31.32 | 25.79 | — |
| MemoChat | 22.09 | 17.18 | — |
| Zep | — | — | 未报告结果 |

**关键洞察**:
1. **Hierarchical Storage 显著优于 Flat**: MemoryOS/MemTree/MemOS (均有分层) 排名前 3
2. **Graph-based Extraction 提升 Information Extraction**: Mem0g/MemTree 在该维度得分最高
3. **Connecting + Transforming 组合是关键**: MemoryOS 同时实现 5 种管理操作，综合表现最佳
