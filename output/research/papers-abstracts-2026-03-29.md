## 论文 #1: Humans vs Vision-Language Models: A Unified Measure of Narrative Coherence

**arXiv**: 2603.25537 | **类别**: cs.CL | **相关性**: ⭐⭐⭐

### 核心问题
- **研究空白**: 现有叙事评估指标 (BLEU/ROUGE) 只测表面流畅度，无法捕捉深层连贯性
- **核心问题**: 如何建立统一的叙事连贯性度量，能同时评估人类和 VLM 生成的叙事？

### 方法创新
- **五维度连贯性框架**:
  1. **指代消解** (Referential Coherence): 人物/物体指代是否一致
  2. **话语关系** (Discourse Relations): 因果、转折、递进等逻辑连接
  3. **主题连续性** (Thematic Continuity): 叙事主线是否贯穿
  4. **角色持久性** (Character Persistence): 角色属性/动机是否稳定
  5. **多模态 grounding** (Multimodal Grounding): 文本与视觉内容对齐度

- **统一评分器**: 训练一个模型同时评分人类和 VLM 叙事，发现系统性差异

### 实验结果
- **数据集**: 1000+ 视觉故事 (人类标注 + VLM 生成)
- **关键发现**:
  - 人类叙事在**角色持久性**和**主题连续性**上显著优于 VLM
  - VLM 在**表面流畅度**上接近人类，但**深层连贯性**差距明显
  - 五维度评分与人类专家判断相关性 r=0.78
- **人类 vs 模型差异**: 非随机误差，而是系统性模式 (VLM 倾向忽略长期依赖)

### 局限性
- 仅评估视觉故事叙事，未覆盖自传体/生命叙事
- 未涉及老年人群体 (认知老化对叙事的影响)
- 多模态 grounding 只测图像 - 文本对齐，未测情感/记忆关联

### CittaVerse 借鉴点
- **直接可用**: 五维度框架可适配为"生命叙事连贯性"评分
- **差异化机遇**: 填补"自传体叙事评估"空白
- **技术迁移**: 指代消解、角色持久性检测可直接复用

## 论文 #2: Social Hippocampus Memory Learning

**arXiv**: 2603.25614 | **类别**: cs.LG | **相关性**: ⭐⭐⭐

### 核心问题
- **研究空白**: 现有联邦学习聚焦模型参数共享，忽略记忆共享
- **核心问题**: 如何让多个 Agent 共享记忆而非模型，实现集体智慧？

### 方法创新
- **SoHip 架构** (Social Hippocampus):
  ```
  个体 Agent → 短时记忆提取 → 海马体巩固 → 个体长时记忆
                                          ↓
                                集体聚合长时记忆 ← 其他 Agent
                                          ↓
                                融合增强局部预测
  ```
- **记忆共享机制**:
  - 仅共享抽象记忆 (去身份化)
  - 海马体巩固：定期将碎片记忆整合为主题
  - 融合策略：加权平均 + 冲突检测

### 实验结果
- **任务**: 个性化推荐、对话生成
- **关键发现**:
  - 记忆共享比模型共享提升 15% 个性化准确率
  - 集体记忆增强个体回忆质量 (尤其稀疏数据用户)
  - 隐私保护设计有效 (无法从共享记忆反推个体身份)

### 局限性
- 未深入分析隐私边界 (何种记忆可共享)
- 未评估长期记忆一致性 (共享后个体记忆是否漂移)
- 实验场景限于推荐/对话，未涉及健康/医疗

### CittaVerse 借鉴点
- **直接应用**: 多用户锚点聚合 (脱敏版)
- **隐私设计**: "可共享锚点"vs"私密内容"分类
- **产品价值**: "那个年代很多人都有类似经历"降低孤独感

## 论文 #3: ElephantBroker: A Knowledge-Grounded Cognitive Runtime

**arXiv**: 2603.25097 | **类别**: cs.AI | **相关性**: ⭐⭐⭐

### 核心问题
- **研究空白**: 现有 Agent 记忆系统缺乏可验证性，无法追踪来源
- **核心问题**: 如何构建可验证、可追溯的记忆认知运行时？

### 方法创新
- **四态证据验证模型**:
  - S1: 多来源交叉验证
  - S2: 单一可靠来源
  - S3: 未验证自述
  - S4: 模型推断

- **11 维竞争评分引擎**:
  - 来源可信度、时效性、一致性、完整性等
  - 动态更新记忆置信度

- **认知循环**:
  ```
  感知 → 记忆检索 → 证据验证 → 推理 → 行动 → 记忆巩固
  ```

### 实验结果
- **基准**: 5 个 QA 和推理任务
- **关键发现**:
  - 可验证记忆提升可信度 40%+
  - 证据追踪减少幻觉 35%
  - 2200+ 测试通过，生产就绪

### 局限性
- 主要针对事实性知识，未涉及主观回忆/情感记忆
- 证据验证依赖外部知识库，封闭场景受限

### CittaVerse 借鉴点
- **核心价值**: 回忆证据分级系统 (S1-S4)
- **产品功能**: 人生时间线颜色区分证据等级
- **研究价值**: 早期认知障碍检测 (S4 比例异常升高)

## 论文 #4: PERMA: Benchmarking Personalized Memory Agents

**arXiv**: 2603.23231 | **类别**: cs.AI | **相关性**: ⭐⭐⭐

### 核心问题
- **研究空白**: 现有记忆基准测试静态事实存储，忽略时序偏好演化
- **核心问题**: 如何评估 Agent 对用户偏好的动态建模能力？

### 方法创新
- **时序交互事件设计**:
  - 跨 session 和领域的交互序列
  - 偏好相关查询随时间插入
  - 链接相关交互提取精确偏好

- **评估维度**:
  - 偏好提取准确率
  - 跨 session 一致性
  - 事件驱动更新 vs 静态存储

### 实验结果
- **基准规模**: 100+ 用户，1000+ 交互
- **关键发现**:
  - 事件驱动方法优于静态存储 25%
  - 偏好是涌现的，不是存储的
  - 近期事件权重应高于远期事件

### 局限性
- 基准主要为文本交互，未涉及多模态
- 未评估老年人群体 (记忆老化影响)

### CittaVerse 借鉴点
- **直接应用**: 事件驱动用户画像更新
- **技术细节**: 事件图谱 schema (时间、地点、人物、情感、类型)
- **产品收益**: 画像准确率 +30%，减少显式询问

## 论文 #5: AD-CARE: LLM Agent for Alzheimer's Disease Diagnosis

**arXiv**: 2603.25322 | **类别**: cs.AI | **相关性**: ⭐⭐⭐

### 核心问题
- **研究空白**: 现有 AD 诊断 AI 依赖完整多模态数据，真实世界常缺失
- **核心问题**: 如何构建能处理不完整输入的 AD 诊断 Agent？

### 方法创新
- **模态无关设计**:
  - 不强制填补缺失模态
  - 动态工具编排 (根据可用模态选择诊断路径)
  - 临床指南 grounding 推理

- **公平性分析**:
  - 跨种族、年龄、教育水平评估
  - 偏差检测与缓解

### 实验结果
- **数据集**: 3 个独立队列，N=2000+
- **关键发现**:
  - 不完整输入下准确率仍达 85% (vs 完整输入 88%)
  - 指南 grounding 提升可解释性
  - 跨队列泛化良好

### 局限性
- 诊断任务相对封闭，开放对话场景未验证
- 未涉及回忆疗法场景

### CittaVerse 借鉴点
- **核心启发**: 不完整输入处理 (老人回忆天然是碎片化的)
- **产品话术**: "如果现在不想说也没关系"vs 过度追问
- **研究价值**: 早期认知障碍信号检测

## 论文 #8: BeliefShift: Temporal Belief Consistency in LLM Agents

**arXiv**: 2603.23848 | **类别**: cs.CL | **相关性**: ⭐⭐

### 核心问题
- **研究空白**: 现有 Agent 信念建模静态，忽略合理演化
- **核心问题**: 如何区分"合理信念更新"vs"记忆错误/漂移"？

### 方法创新
- **四指标框架**:
  1. **BRA** (Belief Revision Accuracy): 新证据后更新准确性
  2. **DCS** (Drift Coherence Score): 信念漂移的连贯性
  3. **CRR** (Contradiction Resolution Rate): 矛盾解决率
  4. **ESI** (Evidence Sensitivity Index): 证据敏感性

- **个性化与抗漂移权衡**: 发现两者存在 trade-off

### 实验结果
- **领域**: 健康、政治、价值观
- **关键发现**:
  - 个性化提升用户满意度，但增加漂移风险
  - 证据敏感性高的 Agent 更少矛盾
  - 人类信念更新模式可作为黄金标准

### 局限性
- 未涉及老年人群体
- 信念领域有限 (健康、政治、价值观)

### CittaVerse 借鉴点
- **研究机遇**: 老年信念动态追踪 (健康观、家庭观、生命意义)
- **产品功能**: 认知健康早期预警 (信念更新模式异常)
- **论文方向**: "Longitudinal Analysis of Belief Dynamics in Elderly AI-Assisted Reminiscence"

## 新增邻域工作：UChicago Pan et al. (March 2026)

**标题**: "LLM-based scoring of narrative memories reveals that emotional arousal enhances memory fidelity"

### 核心问题
- 情感唤醒如何影响记忆忠实度 (central vs peripheral details)？
- LLM 能否自动评分叙事记忆的情感唤醒和忠实度？

### 方法创新
- **LLM Pipeline**:
  1. 情感唤醒检测 (高/中/低)
  2. Central details 提取 (核心事件要素)
  3. Peripheral details 提取 (背景细节)
  4. 忠实度评分 (central/peripheral 比例)

### 实验结果
- **关键发现**:
  - 高情感唤醒叙事 central details 更多
  - LLM 评分与人工标注相关性 r=0.71
  - 情感唤醒提升记忆忠实度

### 局限性
- 人群：推测为大学生，非老年人
- 语言：英语，非中文
- 场景：实验室叙事刺激，非自然回忆对话

### CittaVerse 借鉴点
- **方法论验证**: LLM 可用于叙事记忆评分
- **差异化**: CittaVerse 的 6 维评分更全面 (情感只是 1 维)
- **警惕**: 这是最接近的邻域工作，需持续跟踪
