1. ★★★ TraceMem — 叙事记忆图谱从对话痕迹中构建 (arXiv:2602.09712, 2026-02)
   链接：https://arxiv.org/abs/2602.09712
   核心：
   - 提出 TraceMem 框架，从用户对话痕迹中编织结构化叙事记忆图谱
   - 三阶段流程：
     (1) 短时记忆处理：演绎式主题分割，提取语义表征
     (2) 突触记忆巩固：将片段总结为情景记忆，蒸馏为用户特定痕迹
     (3) 系统记忆巩固：两阶段层次聚类，组织为连贯叙事线索
   - 输出结构化用户记忆卡片 (user memory cards)
   - 在 LoCoMo benchmark 上达到 SOTA，超越基线在多跳和时序推理任务
   VSNC 启示：
   - **与阿宝"人生故事书"愿景高度契合** — 都是从对话中构建叙事记忆
   - "narrative memory schemata"概念可直接用于阿宝的长期记忆组织
   - 层次聚类方法可参考用于阿宝的故事章节自动划分
   - LoCoMo benchmark 可作为阿宝叙事质量的评估参照
   验证等级：V2（arXiv 确认，摘要信息清晰）

2. ★★☆ TwinVoice — LLM 人格模拟的多维基准 (arXiv:2510.25536, 2025-10)
   链接：https://arxiv.org/abs/2510.25536
   核心：
   - 提出 TwinVoice 基准，评估 LLM 的人格模拟能力
   - 三个维度：Social Persona（公共社交）、Interpersonal Persona（私密对话）、
     Narrative Persona（角色表达）
   - 六项基础能力评估：opinion consistency, memory recall, logical reasoning,
     lexical fidelity, persona tone, syntactic style
   - 发现：先进模型在 syntactic style 和 memory recall 上仍显著低于人类基线
   VSNC 启示：
   - **六项能力可直接用于阿宝的叙事质量评分维度设计**
   - "memory recall"评估方法与阿宝的回忆引导质量评估相关
   - "Narrative Persona"维度与阿宝帮助用户构建连贯人生叙事的目标一致
   - 人类基线数据可作为阿宝评分系统的参考锚点
   验证等级：V2（arXiv 确认）

3. ★★☆ ComoRAG — 认知启发的记忆组织 RAG (arXiv:2508.10419, 2025-08)
   链接：https://arxiv.org/abs/2508.10419（AAAI 2026 接收）
   核心：
   - 针对长叙事推理的认知启发式记忆组织 RAG
   - 解决长故事/小说理解中的复杂情节线和实体关系纠缠问题
   - 记忆组织策略参考人类认知机制
   VSNC 启示：
   - RAG + 记忆组织的结合可参考用于阿宝的长时记忆检索
   - 认知启发式设计原则与 TraceMem 的脑启发架构相呼应
   验证等级：V1（arXiv 摘要确认，AAAI 2026 接收）

4. ★★☆ Narrative Memory in Machines — 多 Agent 叙事弧提取 (arXiv:2508.07010, 2025-08)
   链接：https://arxiv.org/abs/2508.07010
   核心：
   - 多 Agent 系统用于提取和分析 serialized TV 叙事中的故事弧
   - 解决时间分布式情节的复杂分析挑战
   VSNC 启示：
   - **多 Agent 叙事分析方法**可参考用于阿宝的人生故事章节划分
   - 与 Sophia 框架（#24）的多 Agent 设计形成呼应
   验证等级：V1（arXiv 摘要确认）
