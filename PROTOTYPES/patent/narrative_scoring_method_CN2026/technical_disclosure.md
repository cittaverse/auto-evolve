## 一、技术背景

### 1.1 现有技术局限

回忆疗法（Reminiscence Therapy）在老年认知干预中已被广泛验证有效。然而，现有技术方案存在以下核心问题：

**问题 1：评估方法主观性强**
- 传统叙事质量评估依赖人类专家人工评分，一致性差（Inter-rater reliability < 0.6）
- 现有自动化方法（如连接词频率计数）对老年口语失效——口语不按书面逻辑组织

**问题 2：缺乏引导反馈闭环**
- 现有 AI 系统仅做"检测/分类"（如：是否有认知障碍），不提供"如何提升叙事质量"的可操作建议
- 无法形成"采集→评估→引导→再采集"的正向增强循环

**问题 3：中文老年语料优化缺失**
- 现有研究（如 JMIR Aging 2025、Frontiers 2025）均基于英文语料
- 中文口语的省略、倒装、方言特征导致直接迁移失效

### 1.2 最接近的现有技术

| 技术 | 公开号/来源 | 缺陷 |
|------|-------------|------|
| Google 内容摘要系统 | WO2025042780A1 | 仅做摘要生成，无质量评分 |
| MemGraph 专利匹配 | arXiv 2025 | 用于专利检索，非叙事评估 |
| AI 生命故事结构化 | JMIR Aging 2025 | 仅技术可行性，无评分算法细节 |
| 字符级语言生物标志物 | Frontiers Aging Neurosci 2025 | 仅检测阿尔茨海默，无叙事质量引导 |

## 二、发明内容

### 2.1 核心创新点

本发明提出一种**神经符号混合架构**的叙事质量评估方法，将 LLM 的语义理解能力与图论的结构化计分能力结合，实现：

1. **客观化评分**：拓扑连贯性使用数学公式计算，消除主观偏差
2. **可解释反馈**：5 维度评分 + 自然语言理由，直接用于引导用户补充细节
3. **中文优化**：针对中文老年口语特征（省略、倒装、方言）进行 LLM 微调

### 2.2 技术方案

#### 方法流程（对应独立权利要求 1）

```
步骤 S1：接收非结构化口述文本
        ↓
步骤 S2：LLM 事件提取 → 输出 JSON 事件图谱
        （节点：事件；边：因果/时序关系）
        ↓
步骤 S3：图论结构计分 → 拓扑连贯性 + 事件密度
        ↓
步骤 S4：LLM 多维度内容评分 → 5 维度打分 + 理由
        ↓
步骤 S5：加权融合 → 综合等级（S/A/B/C）+ 引导反馈
```

#### 事件图谱结构（对应从属权利要求）

```json
{
  "events": [
    {
      "id": "E1",
      "text": "8 岁那年夏天，我第一次去杭州",
      "timestamp": "1952-07",

      "participants": ["我", "父亲"],
      "emotions": ["兴奋", "好奇"]
    },
    {
      "id": "E2",
      "text": "我们在西湖边拍了一张黑白照片",
      "timestamp": "1952-07",
      "location": "西湖",

      "emotions": ["快乐"]
    }

  "relations": [
    {
      "from": "E1",
      "to": "E2",
      "type": "temporal",  // temporal / causal / contrastive
      "confidence": 0.92
    }
  ]

```

#### 图论计分公式（对应从属权利要求）

**拓扑连贯性**（Topology Coherence）：
```
TC = (有效因果边数 / (事件数 - 1)) × 100
```
- 有效因果边：type ∈ {causal, temporal} 且 confidence > 0.7
- 理想树状结构：N 个事件应有 N-1 条边

**事件密度**（Event Density）：
```
ED = min(事件数 / 字数 × 1000, 100)
```
- 归一化到每千字事件数，上限 100 分

#### 5 维度内容评分（对应从属权利要求 2）

| 维度 | 定义 | 评分标准 |
|------|------|----------|
| 叙事连贯性 | 故事是否有清晰起承转合 | 1-5 分，LLM 判断 |
| 内部细节 | 感官细节（视觉/听觉/嗅觉/触觉） | 1-5 分，计数 + 质量 |
| 外部细节 | 时间/地点/人物等情境信息 | 1-5 分，计数 |
| 情感深度 | 情绪表达的具体程度 | 1-5 分，是否具体描述感受 |
| 自我参照 | 是否有反思/意义建构 | 1-5 分，是否包含"我意识到/我明白了" |

#### 综合等级映射（对应从属权利要求 3）

```
综合得分 = TC × 0.3 + ED × 0.1 + 内容分 × 0.6

S 级 (90-100): 可直接用于出版，无需编辑
A 级 (75-89):  需轻度编辑润色
B 级 (60-74):  需引导补充细节（当前默认等级）
C 级 (<60):    需重新采集或深度访谈
```

#### 引导反馈生成（对应从属权利要求 3）

根据最低分维度动态生成：

```
IF 内部细节 < 3:
  → "您能描述一下当时的声音/气味/触感吗？比如空气中有什么味道？"
  
IF 外部细节 < 3:
  → "这件事发生在哪一年？当时您在哪里？还有谁在场？"
  
IF 情感深度 < 3:
  → "当时您心里是什么感受？是紧张、兴奋，还是其他情绪？"
  
IF 自我参照 < 3:
  → "这件事对您后来的人生产生了什么影响？您从中学到了什么？"
```

### 3.2 代码实现（核心片段）

```python
# pipeline_v0.3.py - 核心计分逻辑

def calculate_topology_coherence(events, relations):
    """计算拓扑连贯性"""
    n_events = len(events)
    if n_events <= 1:
        return 100.0 if n_events == 1 else 0.0
    
    valid_edges = sum(
        1 for r in relations 
        if r['type'] in ['causal', 'temporal'] and r['confidence'] > 0.7
    )
    
    # 理想树状结构应有 N-1 条边
    ideal_edges = n_events - 1
    return min((valid_edges / ideal_edges) * 100, 100)


def calculate_content_scores(narrative_text, llm_client):
    """LLM 多维度内容评分"""
    prompt = f"""
    请对以下老年人口述叙事进行 5 维度评分（每项 1-5 分）：
    
    1. 叙事连贯性：故事是否有清晰起承转合
    2. 内部细节：感官细节（视觉/听觉/嗅觉/触觉）
    3. 外部细节：时间/地点/人物等情境信息
    4. 情感深度：情绪表达的具体程度
    5. 自我参照：是否有反思/意义建构
    
    输出格式：
    {{
      "narrative_coherence": {{ "score": 4, "reason": "..." }},
      "internal_details": {{ "score": 2, "reason": "缺乏感官细节" }},
      "external_details": {{ "score": 3, "reason": "..." }},
      "emotional_depth": {{ "score": 3, "reason": "..." }},
      "self_reference": {{ "score": 4, "reason": "..." }}
    }}
    
    口述内容：
    {narrative_text}
    """
    
    response = llm_client.chat(prompt)
    return json.loads(response)
```

## 五、权利要求建议（v1.1 - B1 v2.0 对齐）

### 独立权利要求 1（方法）- 场景特异性版本
一种基于怀旧对话场景的中文老年认知特征被动采集方法，包括：
- 在怀旧对话交互过程中，非侵入式采集老年人口述语音数据；
- 对所述语音数据进行中文方言优化处理，适配普通话及地方方言的声学特征；
- 使用大语言模型从所述口述语音的转写文本中提取事件图谱，所述事件图谱包含多个事件节点及节点间的因果/时序边；
- 使用图论算法计算所述事件图谱的拓扑连贯性得分；
- 使用大语言模型对所述口述文本进行多维度内容评分，包括叙事连贯性、内部细节、外部细节、情感深度和自我参照；
- 基于所述拓扑连贯性得分和所述多维度内容评分，生成 L0 六维度认知特征向量及引导反馈。

**其特征在于**：所述怀旧对话场景为非临床测试环境下的自然对话，所述中文方言优化模块针对声调语言特性进行优化。

### 从属权利要求 2（5 维度细化）
如权利要求 1 所述的方法，其中所述多维度内容评分包括：
- 叙事连贯性：评估故事起承转合的完整性，1-5 分；
- 内部细节：评估感官细节（视觉/听觉/嗅觉/触觉）的丰富程度，1-5 分；
- 外部细节：评估时间/地点/人物等情境信息的完整程度，1-5 分；
- 情感深度：评估情绪表达的具体程度，1-5 分；
- 自我参照：评估反思与意义建构内容的深度，1-5 分。

### 从属权利要求 3（引导反馈生成）
如权利要求 1 所述的方法，其中所述引导反馈根据所述多维度内容评分中的最低分维度动态生成：
- 当内部细节评分低于阈值时，生成提示用户补充感官细节的问题；
- 当外部细节评分低于阈值时，生成提示用户补充时间/地点/人物信息的问题；
- 当情感深度评分低于阈值时，生成提示用户描述具体感受的问题；
- 当自我参照评分低于阈值时，生成提示用户反思事件意义的问题。

### 从属权利要求 4（中文方言优化）
如权利要求 1 所述的方法，其中所述中文方言优化处理包括：
- 针对汉语声调语言特性的声学模型适配；
- 针对地方方言（包括但不限于粤语、吴语、闽南语、川渝话）的语音识别优化；
- 针对老年口语特征（语速慢、停顿多、重复）的预处理模块。

### 独立权利要求 5（系统）
一种基于怀旧对话场景的中文老年认知特征被动采集系统，包括：
- 非侵入式语音采集模块，用于在怀旧对话交互中被动采集语音数据；
- 中文方言优化模块，用于针对普通话及地方方言进行声学适配；
- 事件图谱提取模块，用于从口述叙事中提取事件节点及因果/时序关系；
- 叙事连贯性评分模块，用于使用图论算法计算拓扑连贯性得分；
- 自我参照评估模块，用于评估口述中的反思与意义建构内容；
- 认知特征输出模块，用于输出 L0 六维度认知特征向量。

**其特征在于**：所述系统在非临床测试环境下的自然对话中运行，所述中文方言优化模块针对声调语言特性进行优化。
