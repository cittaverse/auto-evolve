## 使用方式

- 每个目录下有 `INDEX.md`，包含该域的**结论摘要 + 文件清单 + 验证等级**
- 需要原始细节时，回溯 `research/` 对应文件
- 证据扫描周报归入 `04-competitive-intel/`

## 更新规则

- 新研究完成后，先写入 `research/`，再在对应知识域 `INDEX.md` 追加条目
- 跨域研究（如 Rememo 既是竞品又涉及合作）在主域建文件，其他域 INDEX 中加交叉引用
- 每周周报同步更新 `05-product-strategy/INDEX.md`
- 每周整理日志写入 `CONSOLIDATION_LOG_YYYY-MM-DD.md`
