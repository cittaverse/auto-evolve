## Bottom Line

全球 22+ 竞品中，**无产品同时具备"临床验证 + 叙事评分 + 中文优化"三要素**。CittaVerse 的差异化定位 = 临床验证 × 叙事评分 × 中文优化 × 开源策略。Rememo (CHI 2026) 是最近邻但定位互补（前 session 准备 vs 后 session 评估），可合作非竞争。中国市场缺少科研背书的认知健康产品，存在专业壁垒建立机会。

## 核心知识点

### 竞品定位矩阵
```
                    临床导向 ←────→ 消费导向
                         │
            CittaVerse ──┤── Vivid
            Rememo ──────┤
                         │── LifeStory
                         │── StoryFile
                         │── MemoryLane
                         │── StoryWorth
                         │── 小年糕 / 时光小屋
```

### CittaVerse vs Rememo 互补性
| 维度 | Rememo | CittaVerse |
|------|--------|------------|
| 定位 | 会话前准备（生成记忆触发器） | 会话后评估（叙事质量评分） |
| 技术 | 生成式 AI (SDXL/Flux.1) | 神经符号 AI (规则+LLM) |
| 用户 | 治疗师/护理人员 | 研究人员/临床医生 |
| 文化 | 新加坡多元文化 | 中国老年人群体 |
| 输入 | 照片 → 生成问题 | 照片/语音 → 引导回忆 → 评分 |
| 输出 | 会话引导建议 | 叙事质量评分 + 人生故事书 |

### 高价值对标定价参考
- StoryFile: $99-299/月 (高净值家庭)
- MemoryLane: £49 一次性 (普通家庭)
- StoryWorth: $99/年 (中年人为父母订购)
- LifeStory: $199/年 (老年人)
- Remento: $299-999 一次性 (Shark Tank 曝光)
- Eternos: $10.3M Seed 融资，HLM 框架

### 中国市场特征
- 内容/娱乐导向为主（小年糕、时光小屋）
- 缺少临床/研究背书
- 养老产业政策利好（银发经济）
- 专业壁垒建立窗口期

### 新增高相关论文 (03-28)

**Amory** (arXiv 2601.06282):
- 叙事驱动的记忆框架，通过 agentic reasoning 构建结构化记忆
- 连贯性驱动检索优于基于嵌入方法
- **与 CittaVerse 互补非竞争**，叙事结构价值独立验证
- **用途**: 论文 Discussion 可引用，支持"叙事结构作为记忆组织核心"的论点

**SpeechCARE** (arXiv 2511.08132):
- NIA PREPARE 挑战赛方案，Whisper+MoE 架构
- 早期认知障碍检测达到 SOTA 水平
- **语音模态验证**，CittaVerse 多模态整合应加速
- **用途**: 论文 Methods 可参考多模态融合架构
