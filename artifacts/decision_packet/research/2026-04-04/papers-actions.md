## 二、P0 优先级 (立即实施，1-4 周)

### 行动 1: 实现分层记忆架构

**灵感来源**: 2604.01705 (记忆统一框架), 2604.02280 (自适应遗忘)

**具体行动**:
```
当前状态 → 目标状态
单一记忆存储 → 三层记忆架构:
  ├─ 工作记忆 (当前对话上下文，session 级)
  ├─ 短期记忆 (7 天内交互，带遗忘曲线)
  └─ 长期记忆 (核心人生故事，永久存储)
```

**技术要点**:
- 为每个记忆片段添加 `memory_type` 标签
- 实现基于 Ebbinghaus 曲线的自动降级 (短期→长期/遗忘)
- 记忆检索时按类型加权

**预期收益**:
- 减少无关记忆干扰，提升叙事连贯性
- 模拟人类记忆特性，增强用户感知自然度

**负责人**: Core (工程实现)  
**验证指标**: 叙事连贯性评分提升 15%+

### 行动 2: 实现人格引导的叙事引导

**灵感来源**: 2604.01671 (PRCCF 人格引导检索)

**具体行动**:
```python
# 伪代码示例
class PersonaGuidedNarrative:
    def __init__(self, user_profile):
        self.user_profile = user_profile  # 包含：内向/外向、怀旧倾向、情感表达方式
    
    def generate_prompt(self, memory_candidates):
        # 联合建模语义兼容性 + 人格对齐
        semantic_score = self.semantic_similarity(memory_candidates)
        persona_score = self.persona_alignment(memory_candidates, self.user_profile)
        return weighted_ranking(semantic_score, persona_score)
```

**人格维度建议**:
| 维度 | 取值 | 引导策略 |
|------|------|----------|
| 社交倾向 | 内向/外向 | 内向：深度追问；外向：广度探索 |
| 怀旧倾向 | 高/中/低 | 高：主动触发老照片/老歌；低：聚焦当下 |
| 情感表达 | 直接/含蓄 | 直接：情感词丰富；含蓄：间接引导 |

**预期收益**:
- 用户感知"这个 AI 懂我"
- 叙事深度提升

**负责人**: Core + Hulk (人格模型设计)  
**验证指标**: 用户满意度 NPS +10

### 行动 3: 构建叙事质量评估内部基准

**灵感来源**: 2604.01993 (SAFE 原子反馈), 2604.02280 (LOCOMO/LOCCO 基准)

**具体行动**:
```
叙事质量评估维度 (原子化):
├─ 内容完整性 (0-5 分)
│   ├─ 时间线清晰度
│   ├─ 因果链完整性
│   └─ 关键细节丰富度
├─ 情感表达 (0-5 分)
│   ├─ 情感词汇多样性
│   ├─ 情感强度匹配度
│   └─ 情感变化轨迹
├─ 叙事连贯性 (0-5 分)
│   ├─ 话题转换流畅度
│   ├─ 前后一致性
│   └─ 主题聚焦度
└─ 互动质量 (0-5 分)
    ├─ 问题响应相关性
    ├─ 主动追问价值
    └─ 共情表达适当性
```

**实施步骤**:
1. 标注 100 段现有叙事对话 (Hulk 初标 + 人工复核)

3. 集成到产品 dashboard

**预期收益**:
- 量化产品迭代效果
- 识别低质量叙事模式

**负责人**: Hulk (标注规范) + Core (自动化)  
**验证指标**: 评分者间信度 (Cohen's κ) > 0.7

## 三、P1 优先级 (短期规划，1-3 个月)

### 行动 4: 实现自适应记忆遗忘

**灵感来源**: 2604.02280, 2601.03938 (FOREVER)

**具体行动**:
- 为每个记忆片段计算"遗忘分数":
  ```
  forget_score = w1 * recency_decay + w2 * frequency_low + w3 * semantic_drift
  ```
- 高遗忘分数 → 定期复习提醒 ("您之前提到过...想再聊聊吗？")
- 极高遗忘分数 → 归档 (不删除，但降低检索权重)

**预期收益**:
- 减少记忆存储成本 30%+
- 提升记忆检索相关性
