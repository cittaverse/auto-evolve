### 3. Benchmark 标注方案 (`benchmark-annotation-protocol.md`)

**文件**: `research/paper/benchmark-annotation-protocol.md`  
**大小**: 7.4KB  
**内容**:

#### 3.1 Benchmark 数据集设计

**目标**: 创建包含 100 段老年叙事的 Benchmark 数据集，用于验证 LLM 六维评分与人工评分的一致性。

**数据来源**:
- Pilot RCT 基线叙事 (N=60 人 × 1 段 = 60 段)
- 补充招募叙事 (N=20 人 × 2 段 = 40 段)
- 总计: 100 段

**叙事主题** (平衡分布):
| 主题 | 数量 | 示例提示 |
|------|------|----------|
| 童年记忆 | 20 | "请讲述一件您童年印象最深刻的事" |
| 求学经历 | 15 | "请讲述您的求学故事" |
| 工作经历 | 15 | "请讲述您的职业生涯" |
| 婚姻家庭 | 20 | "请讲述您结婚/成家的故事" |
| 养育子女 | 15 | "请讲述养育孩子的经历" |
| 退休生活 | 15 | "请讲述退休后的生活" |

**叙事长度**:
- 目标: 每段 300-800 字
- 最短: ≥150 字
- 最长: ≤1500 字

#### 3.2 人工标注流程

**标注员资质**:
- 心理学/护理学/社会工作专业本科及以上
- 经过 6 小时培训 (理论 + 练习)
- ICC 检验达标 (ICC > 0.75)

**培训流程**:
1. 理论培训 (2 小时): 六维定义、评分标准、示例讲解
2. 练习标注 (2 小时): 10 段练习叙事，与金标准对照
3. ICC 检验 (2 小时): 20 段测试叙事，计算 ICC
4. 达标后上岗，未达标者补训

**标注任务**:
每段叙事需标注:
1. L0 六维分数 (各 0-5 分)
2. 事件边界标注 (起始/结束位置)
3. 情感效价标注 (积极/中性/消极 + 强度)
4. 感官细节标注 (五感类别)
5. 社会联结标注 (关系类型 + 互动深度)
6. 自我反思标注 (反思类型)
7. 整体质量评分 (0-100 分)
8. 置信度评分 (1-5 分)

**标注工具**:
- 在线标注平台 (自研或 Label Studio)
- 支持高亮、标注、评分、导出
- 双人独立标注 + 仲裁机制

**质量控制**:
- 10% 叙事双人标注 (计算 ICC)
- 每周质控会议 (讨论分歧案例)
- 标注员间 ICC < 0.75 时重新培训

#### 3.3 标注时间表

| 阶段 | 时间 | 任务 | 产出 |
|------|------|------|------|
| 准备期 | 2026-04-01 ~ 04-10 | 招募标注员、准备培训材料 | 标注员 3 名 |
| 培训期 | 2026-04-11 ~ 04-20 | 理论 + 练习 + ICC 检验 | ICC 报告 |
| 标注期 | 2026-04-21 ~ 05-15 | 100 段叙事标注 | 标注数据集 v1.0 |

#### 3.4 成功标准

| 指标 | 目标值 | 验证方式 |
|------|--------|----------|

**内容**:

#### 4.1 培训目标

培训结束后，评估者能够:
1. 准确理解六维评分的定义和标准
2. 独立对叙事进行六维评分 (ICC > 0.90)
3. 识别常见评分偏差并纠正
4. 使用标注工具完成评分任务

#### 4.2 培训内容

**模块 1: 理论基础 (30 分钟)**
- 自传体记忆与叙事心理学简介
- 六维评分的理论依据
- 评分与心理健康结果的关联

**模块 2: 六维详解 (60 分钟)**
- 每维度的定义、评分标准、示例
- 边界案例讨论
- 常见错误与纠正

**模块 3: 练习评分 (60 分钟)**
- 10 段练习叙事
- 与金标准对照
- 讨论分歧点

**模块 4: ICC 检验 (30 分钟)**
- 20 段测试叙事
- 计算 ICC
- 达标判定 (ICC > 0.90)

#### 4.3 六维速查卡 (可打印)

```
┌─────────────────────────────────────────────────────┐
│  CittaVerse 六维评分速查卡                          │
├─────────────────────────────────────────────────────┤
│                                                     │
│  1. 时间连贯性 (0-5 分)                             │
│  0=无时间信息  1=模糊  2=相对  3=年代  4=精确  5=网络│
│                                                     │
│  2. 情感效价 (0-5 分)                               │
│  基础 2 分 + 复杂度 0-2 分 + 平衡 bonus 1 分          │
│                                                     │
│  3. 感官细节 (0-5 分)                               │
│  0=无  1=视觉  2=视听  3=+嗅/味/触  4=四类  5=五感  │
│                                                     │
│  4. 社会联结 (0-5 分)                               │
│  多样性 0-2 + 互动深度 0-2 + 情感纽带 0-1           │
│                                                     │
│  5. 事件分段 (0-5 分)                               │
│  0=流水账  2=时间/地点转换  4=因果联系  5=层次化    │
│                                                     │
│  6. 自我反思 (0-5 分)                               │
│  因果/成长/价值/整合，每类 +1 分，上限 5 分           │
│                                                     │
│  总分 = Σ(权重 × 维度分)                            │
│  权重：情感 0.25, 社会 0.25, 时间 0.12,             │
│        感官 0.13, 事件 0.12, 反思 0.13              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### 4.4 ICC 检验流程

1. 20 段测试叙事 (覆盖各主题/长度/质量)
2. 3 名评估者独立评分
3. 计算组内相关系数 (ICC, two-way random, absolute agreement)
4. 判定标准:
   - ICC ≥ 0.90: 优秀，可上岗
   - 0.75 ≤ ICC < 0.90: 良好，补训后上岗
   - ICC < 0.75: 不合格，重新培训
