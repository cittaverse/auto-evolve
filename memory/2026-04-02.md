### 核心发现

1. **AD-CARE 多队列评估设计迁移**
   - 三队列评估 → 叙事类型分层 (创伤性/正向/中性/反思性)
   - 公平性分析 → 代际/方言分层
   - 指南驱动 → 心理学理论驱动

2. **Reward Hacking 防御机制**
   - 攻击向量：关键词堆砌/情感词滥用/虚构细节/模板化叙述
   - 防御机制：信息密度检测/情感 - 事件一致性检验/内部一致性检验/叙事多样性检测

3. **Over-Verification 陷阱规避**
   - 分层验证：L0 快速过滤 → L1 边界案例仲裁 → L2 高价值人工复核

   - 性能目标：80% 样本 L0 快速评分，20% 进入 L1

4. **多 Agent 协作架构**
   - 选择：中心化仲裁 (与 MediHive 去中心化差异化)
   - 决策机制：LLM 仲裁层最终裁决 (避免"平庸共识")
