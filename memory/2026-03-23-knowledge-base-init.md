## 设计决策
- **不复制原文**：INDEX 只存压缩结论 + 文件指针，避免内容重复
- **验证等级标注**：每条结论标注 V0-V4 等级
- **交叉引用**：跨域内容（如 Rememo 同时涉及竞品和合作）在主域建条目，其他域 INDEX 引用
- **增量更新规则**：新研究先写 research/，再在对应域 INDEX 追加条目
