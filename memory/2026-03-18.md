## Free Academic Paper APIs (No API Key Required)

### Semantic Scholar

- **URL**: https://www.semanticscholar.org/
- **API**: Free API for developers (new & improved)
- **Features**:
  - AI-powered research tool
  - Extensive database of scientific papers
  - No API key required for core tools
  - Some advanced tools require free registration
- **Use Cases**: Academic paper search, citation analysis, literature review

### arXiv

- **Direct Access**: https://arxiv.org/
- **API**: arXiv API available
- **Features**:
  - Preprint server for physics, mathematics, computer science, etc.
  - No registration required for basic access
  - OAI-PMH interface for bulk harvesting

### PubMed

- **URL**: https://pubmed.ncbi.nlm.nih.gov/
- **API**: E-utilities API (free, requires API key for high volume)
- **Features**:
  - Biomedical literature database
  - 30+ million citations
  - Free for basic use

### Ocean of Papers

- **URL**: https://oceanofpapers.com/
- **Features**:
  - Free academic search engine
  - Aggregates: OpenAlex, PubMed, arXiv, bioRxiv, medRxiv
  - Alternative to Google Scholar

### MCP Servers

- **Semantic Scholar MCP Server** (lobehub.com/mcp/fujishigetemma-semantic-scholar-mcp)
  - No API key required for core tools
  - Some tools require free registration
  - Model Context Protocol integration
