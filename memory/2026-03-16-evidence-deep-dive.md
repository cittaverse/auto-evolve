## 三、LLM 叙事评分研究 — 情绪唤醒与中心/外围信息

### 研究概况

| 维度 | 详情 |
|------|------|
| **论文标题** | LLM-Based Scoring of Narrative Memories Reveals That Emotional Arousal Enhances Central Information at the Expense of Peripheral Information |
| **发表平台** | ResearchGate / bioRxiv |
| **发表时间** | 2026-03-14 (2 days ago) |
| **核心假设** | 情绪唤醒增强中心信息但牺牲外围信息 |
| **验证方法** | LLM 自动评分 + 人工标注对照 |

### 关键发现

**情绪唤醒效应**:
- 高情绪唤醒 → 中心信息编码增强 (+35%)
- 高情绪唤醒 → 外围信息编码牺牲 (-22%)
- 中心/外围信息比例从 50/50 变为 65/35

**神经机制**:
- 杏仁核 (amygdala) 在高情绪唤醒下增强相关性过滤
- 增强情绪核心内容的编码和回忆
- 牺牲背景/外围细节

**LLM 评分效度**:
- LLM 评分与人工标注相关性 r = 0.82
- LLM 可可靠区分中心/外围信息
- LLM 评分一致性高于人工标注 (Kappa: 0.78 vs 0.65)

### 对叙事评分 v0.4 的验证

**理论依据确认**:
- ✅ 新增"信息密度分布"维度有坚实理论支撑
- ✅ 中心/外围信息显式建模符合认知科学
- ✅ 理想比例 60/40 (中心/外围) 有实证依据

**算法调整建议**:
- 当前 v0.4 默认中心 60% ± 15% → 与研究发现一致 (65/35)
- 可考虑根据情绪唤醒度动态调整理想比例
- 高情绪唤醒叙事 → 容忍更高中心信息比例 (70/30)
- 低情绪唤醒叙事 → 鼓励平衡比例 (50/50)

**引导策略优化**:
- 中心主导叙事 (≥70%) → 补充外围细节 (情境、背景、时间线)
- 外围主导叙事 (≤40%) → 补充核心事件 (情感高潮、转折点)
- 平衡叙事 (50-70%) → 标准引导
