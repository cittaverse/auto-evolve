## 研究洞察

### 伦理审批材料的核心要素

**发现**: 中国伦理委员会审查关注以下核心点

1. **风险受益比**: 必须明确说明风险等级和缓解措施
2. **知情同意**: 弱势群体 (MCI 老年人) 需要特殊保护
3. **数据隐私**: 加密存储、去标识化、访问控制
4. **研究者资质**: GCP 培训证书是硬性要求
5. **科学价值**: 需要说明前期研究基础和创新点

**产品启示**:
- 伦理审批不是障碍，是产品质量保障
- 提前准备伦理材料，避免后期返工
- 与伦理委员会保持沟通，及时响应修改意见

### Pilot RCT 的关键成功因素

**基于文献和前期经验**:

1. **招募速度**: 目标 80 人，建议 2-3 周内完成
2. **依从性保障**: 每日提醒 + 简化操作 + 技术支持
3. **数据质量**: 双人录入 + 自动化校验 + 定期监控
4. **不良事件管理**: 24 小时报告机制 + 应急预案

**风险点**:
- 招募不足 (目标 80 人，实际可能<60)
- 依从性低 (目标≥70%，实际可能<50%)
- 技术故障 (ASR API 不稳定)

**缓解措施**:
- 多渠道招募 (社区 + 医院 + 线上)
- 简化操作流程 (一键录音 + 自动上传)
- 备用方案 (mock ASR + 人工转写)
