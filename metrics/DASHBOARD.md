## 📝 使用说明

### 手动更新指标

```bash
# 每周追踪 (生成 summary-YYYY-MM-DD.md)
./scripts/track-metrics.sh weekly

# 每月汇总 (生成 monthly-YYYY-MM.md)
./scripts/track-metrics.sh monthly

# 抓取单个仓库统计 (需 GITHUB_TOKEN)
./scripts/fetch-github-stats.sh cittaverse/pipeline

# 更新 DASHBOARD.md
# 手动编辑 metrics/DASHBOARD.md 填入最新数据
```

### 自动化 (建议 cron)

```cron
# 每周日 09:00 UTC 执行追踪
0 9 * * 0 cd /home/node/.openclaw/workspace-hulk && ./scripts/track-metrics.sh weekly

# 每月 1 日 09:00 UTC 执行月度汇总
0 9 1 * * cd /home/node/.openclaw/workspace-hulk && ./scripts/track-metrics.sh monthly
```

### GitHub API Token 配置

```bash
# 获取 Token: https://github.com/settings/tokens
# 权限：repo (读取即可)
export GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxx
```
